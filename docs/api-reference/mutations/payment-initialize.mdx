Initializes payment process when it is required by gateway.

```graphql
paymentInitialize(
  gateway: String!
  paymentData: JSONString
): PaymentInitialize

```

### Arguments

#### `gateway` ([`String!`](/api-reference/scalars/string.mdx))

A gateway name used to initialize the payment.

#### `paymentData` ([`JSONString`](/api-reference/scalars/json-string.mdx))

Client-side generated data required to initialize the payment.

### Type

#### [`PaymentInitialize`](/api-reference/objects/payment-initialize.mdx)

Initializes payment process when it is required by gateway.
