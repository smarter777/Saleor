Adds products to a collection.

```graphql
collectionAddProducts(
  collectionId: ID!
  products: [ID]!
): CollectionAddProducts

```

### Arguments

#### `collectionId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a collection.

#### `products` ([`[ID]!`](/api-reference/scalars/id.mdx))

List of product IDs.

### Type

#### [`CollectionAddProducts`](/api-reference/objects/collection-add-products.mdx)

Adds products to a collection.
