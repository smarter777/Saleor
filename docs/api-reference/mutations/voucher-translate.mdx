#### `input` ([`NameTranslationInput!`](/api-reference/inputs/name-translation-input.mdx))

Translation language code.

### Type

#### [`VoucherTranslate`](/api-reference/objects/voucher-translate.mdx)

Creates/Updates translations for Voucher.
