Updates a sale.

```graphql
saleUpdate(
  id: ID!
  input: SaleInput!
): SaleUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a sale to update.

#### `input` ([`SaleInput!`](/api-reference/inputs/sale-input.mdx))

Fields required to update a sale.

### Type

#### [`SaleUpdate`](/api-reference/objects/sale-update.mdx)

Updates a sale.
