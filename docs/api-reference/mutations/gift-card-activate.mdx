Activate a gift card.

```graphql
giftCardActivate(

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a gift card to activate.

### Type

#### [`GiftCardActivate`](/api-reference/objects/gift-card-activate.mdx)

Activate a gift card.
