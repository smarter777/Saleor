Creates a new menu item.

```graphql
menuItemCreate(
  input: MenuItemCreateInput!
): MenuItemCreate

```

### Arguments

#### `input` ([`MenuItemCreateInput!`](/api-reference/inputs/menu-item-create-input.mdx))

Fields required to update a menu item. Only one of `url`, `category`, `page`, `collection` is allowed per item.

### Type

#### [`MenuItemCreate`](/api-reference/objects/menu-item-create.mdx)

Creates a new menu item.
