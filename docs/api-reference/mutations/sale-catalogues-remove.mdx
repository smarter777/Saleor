Removes products, categories, collections from a sale.

```graphql
saleCataloguesRemove(
  id: ID!
  input: CatalogueInput!
): SaleRemoveCatalogues

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a sale.

#### `input` ([`CatalogueInput!`](/api-reference/inputs/catalogue-input.mdx))

Fields required to modify catalogue IDs of sale.

### Type

#### [`SaleRemoveCatalogues`](/api-reference/objects/sale-remove-catalogues.mdx)

Removes products, categories, collections from a sale.
