Update digital content.

```graphql
digitalContentUpdate(
  input: DigitalContentInput!
  variantId: ID!
): DigitalContentUpdate

```

### Arguments

#### `input` ([`DigitalContentInput!`](/api-reference/inputs/digital-content-input.mdx))

Fields required to update a digital content.

#### `variantId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a product variant with digital content to update.

### Type

#### [`DigitalContentUpdate`](/api-reference/objects/digital-content-update.mdx)

Update digital content.
