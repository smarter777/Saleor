Updates a new shipping zone.

```graphql
shippingZoneUpdate(
  id: ID!
  input: ShippingZoneUpdateInput!
): ShippingZoneUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a shipping zone to update.

#### `input` ([`ShippingZoneUpdateInput!`](/api-reference/inputs/shipping-zone-update-input.mdx))

Fields required to update a shipping zone.

### Type

#### [`ShippingZoneUpdate`](/api-reference/objects/shipping-zone-update.mdx)

Updates a new shipping zone.
