Manage shipping method's availability in channels.

```graphql
shippingMethodChannelListingUpdate(
  id: ID!
  input: ShippingMethodChannelListingInput!
): ShippingMethodChannelListingUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a shipping method to update.

#### `input` ([`ShippingMethodChannelListingInput!`](/api-reference/inputs/shipping-method-channel-listing-input.mdx))

Fields required to update shipping method channel listings.

### Type

#### [`ShippingMethodChannelListingUpdate`](/api-reference/objects/shipping-method-channel-listing-update.mdx)

Manage shipping method's availability in channels.
