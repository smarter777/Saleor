Updates an existing customer.

```graphql
customerUpdate(
  id: ID!
  input: CustomerInput!
): CustomerUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a customer to update.

#### `input` ([`CustomerInput!`](/api-reference/inputs/customer-input.mdx))

Fields required to update a customer.

### Type

#### [`CustomerUpdate`](/api-reference/objects/customer-update.mdx)

Updates an existing customer.
