Adds a gift card or a voucher to a checkout.

```graphql
checkoutAddPromoCode(
  checkoutId: ID!
  promoCode: String!
): CheckoutAddPromoCode

```

### Arguments

#### `checkoutId` ([`ID!`](/api-reference/scalars/id.mdx))

Checkout ID.

#### `promoCode` ([`String!`](/api-reference/scalars/string.mdx))

Gift card code or voucher code.

### Type

#### [`CheckoutAddPromoCode`](/api-reference/objects/checkout-add-promo-code.mdx)

Adds a gift card or a voucher to a checkout.
