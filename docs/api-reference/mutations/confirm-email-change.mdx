Confirm the email change of the logged-in user.

```graphql
confirmEmailChange(
  token: String!
): ConfirmEmailChange

```

### Arguments

#### `token` ([`String!`](/api-reference/scalars/string.mdx))

A one-time token required to change the email.

### Type

#### [`ConfirmEmailChange`](/api-reference/objects/confirm-email-change.mdx)

Confirm the email change of the logged-in user.
