Create a new address for the customer.

```graphql
accountAddressCreate(
  input: AddressInput!
  type: AddressTypeEnum
): AccountAddressCreate

```

### Arguments

#### `input` ([`AddressInput!`](/api-reference/inputs/address-input.mdx))

Fields required to create address.

#### `type` ([`AddressTypeEnum`](/api-reference/enums/address-type-enum.mdx))

A type of address. If provided, the new address will be automatically assigned as the customer's default address of that type.

### Type

#### [`AccountAddressCreate`](/api-reference/objects/account-address-create.mdx)

Create a new address for the customer.
