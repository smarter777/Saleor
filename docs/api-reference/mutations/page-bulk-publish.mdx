Publish pages.

```graphql
pageBulkPublish(
  ids: [ID]!
  isPublished: Boolean!
): PageBulkPublish

```

### Arguments

#### `ids` ([`[ID]!`](/api-reference/scalars/id.mdx))

List of page IDs to (un)publish.

#### `isPublished` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Determine if pages will be published or not.

### Type

#### [`PageBulkPublish`](/api-reference/objects/page-bulk-publish.mdx)

Publish pages.
