Update the shop's address. If the `null` value is passed, the currently selected address will be deleted.

```graphql
shopAddressUpdate(
  input: AddressInput
): ShopAddressUpdate

```

### Arguments

#### `input` ([`AddressInput`](/api-reference/inputs/address-input.mdx))

Fields required to update shop address.

### Type

#### [`ShopAddressUpdate`](/api-reference/objects/shop-address-update.mdx)

Update the shop's address. If the `null` value is passed, the currently selected address will be deleted.
