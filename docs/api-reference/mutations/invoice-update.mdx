Updates an invoice.

```graphql
invoiceUpdate(
  id: ID!
  input: UpdateInvoiceInput!
): InvoiceUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an invoice to update.

#### `input` ([`UpdateInvoiceInput!`](/api-reference/inputs/update-invoice-input.mdx))

Fields to use when updating an invoice.

### Type

#### [`InvoiceUpdate`](/api-reference/objects/invoice-update.mdx)

Updates an invoice.
