Deletes a CheckoutLine.

```graphql
checkoutLineDelete(
  checkoutId: ID!
  lineId: ID = 0
): CheckoutLineDelete

```

### Arguments

#### `checkoutId` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the checkout.

#### `lineId` ([`ID`](/api-reference/scalars/id.mdx))

ID of the checkout line to delete.

### Type

#### [`CheckoutLineDelete`](/api-reference/objects/checkout-line-delete.mdx)

Deletes a CheckoutLine.
