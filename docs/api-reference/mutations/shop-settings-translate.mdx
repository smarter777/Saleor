Creates/Updates translations for Shop Settings.

```graphql
shopSettingsTranslate(
  input: ShopSettingsTranslationInput!
  languageCode: LanguageCodeEnum!
): ShopSettingsTranslate

```

### Arguments

#### `input` ([`ShopSettingsTranslationInput!`](/api-reference/inputs/shop-settings-translation-input.mdx))

Fields required to update shop settings translations.

#### `languageCode` ([`LanguageCodeEnum!`](/api-reference/enums/language-code-enum.mdx))

Translation language code.

### Type

#### [`ShopSettingsTranslate`](/api-reference/objects/shop-settings-translate.mdx)

Creates/Updates translations for Shop Settings.
