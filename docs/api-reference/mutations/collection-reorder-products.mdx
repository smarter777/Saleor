Reorder the products of a collection.

```graphql
collectionReorderProducts(
  collectionId: ID!
  moves: [MoveProductInput]!
): CollectionReorderProducts

```

### Arguments

#### `collectionId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a collection.

#### `moves` ([`[MoveProductInput]!`](/api-reference/inputs/move-product-input.mdx))

The collection products position operations.

### Type

#### [`CollectionReorderProducts`](/api-reference/objects/collection-reorder-products.mdx)

Reorder the products of a collection.
