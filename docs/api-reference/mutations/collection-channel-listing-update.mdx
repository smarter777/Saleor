Manage collection's availability in channels.

#### `input` ([`CollectionChannelListingUpdateInput!`](/api-reference/inputs/collection-channel-listing-update-input.mdx))

Fields required to create or update collection channel listings.

#### [`CollectionChannelListingUpdate`](/api-reference/objects/collection-channel-listing-update.mdx)

Manage collection's availability in channels.
