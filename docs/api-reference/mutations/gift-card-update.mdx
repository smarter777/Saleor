Update a gift card.

```graphql
giftCardUpdate(
  id: ID!
  input: GiftCardUpdateInput!
): GiftCardUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a gift card to update.

#### `input` ([`GiftCardUpdateInput!`](/api-reference/inputs/gift-card-update-input.mdx))

Fields required to update a gift card.

### Type

#### [`GiftCardUpdate`](/api-reference/objects/gift-card-update.mdx)

Update a gift card.
