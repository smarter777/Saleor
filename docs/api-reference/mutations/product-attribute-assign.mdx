Assign attributes to a given product type.

```graphql
productAttributeAssign(
  operations: [ProductAttributeAssignInput]!
  productTypeId: ID!
): ProductAttributeAssign

```

### Arguments

#### `operations` ([`[ProductAttributeAssignInput]!`](/api-reference/inputs/product-attribute-assign-input.mdx))

The operations to perform.

#### `productTypeId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the product type to assign the attributes into.

### Type

#### [`ProductAttributeAssign`](/api-reference/objects/product-attribute-assign.mdx)

Assign attributes to a given product type.
