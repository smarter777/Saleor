Adds an authorization key.

```graphql
authorizationKeyAdd(
  input: AuthorizationKeyInput!
  keyType: AuthorizationKeyType!
): AuthorizationKeyAdd

```

### Arguments

#### `input` ([`AuthorizationKeyInput!`](/api-reference/inputs/authorization-key-input.mdx))

Fields required to create an authorization key.

#### `keyType` ([`AuthorizationKeyType!`](/api-reference/enums/authorization-key-type.mdx))

Type of an authorization key to add.

### Type

#### [`AuthorizationKeyAdd`](/api-reference/objects/authorization-key-add.mdx)

Adds an authorization key.
