Updates an existing staff user.

```graphql
staffUpdate(
  id: ID!
  input: StaffUpdateInput!
): StaffUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a staff user to update.

#### `input` ([`StaffUpdateInput!`](/api-reference/inputs/staff-update-input.mdx))

Fields required to update a staff user.

### Type

#### [`StaffUpdate`](/api-reference/objects/staff-update.mdx)

Updates an existing staff user.
