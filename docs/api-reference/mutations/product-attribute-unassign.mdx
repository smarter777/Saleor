Un-assign attributes from a given product type.

```graphql
productAttributeUnassign(
  attributeIds: [ID]!
  productTypeId: ID!
): ProductAttributeUnassign

```

### Arguments

#### `attributeIds` ([`[ID]!`](/api-reference/scalars/id.mdx))

The IDs of the attributes to unassign.

#### `productTypeId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the product type from which the attributes should be unassigned.

### Type

#### [`ProductAttributeUnassign`](/api-reference/objects/product-attribute-unassign.mdx)

Un-assign attributes from a given product type.
