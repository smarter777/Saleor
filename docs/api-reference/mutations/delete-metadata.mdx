Delete metadata of an object.

```graphql
deleteMetadata(
  id: ID!
  keys: [String!]!
): DeleteMetadata

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an object to update.

#### `keys` ([`[String!]!`](/api-reference/scalars/string.mdx))

Metadata keys to delete.

### Type

#### [`DeleteMetadata`](/api-reference/objects/delete-metadata.mdx)

Delete metadata of an object.
