Updates an existing app.

```graphql
appUpdate(
  id: ID!
  input: AppInput!
): AppUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an app to update.

#### `input` ([`AppInput!`](/api-reference/inputs/app-input.mdx))

Fields required to update an existing app.

### Type

#### [`AppUpdate`](/api-reference/objects/app-update.mdx)

Updates an existing app.
