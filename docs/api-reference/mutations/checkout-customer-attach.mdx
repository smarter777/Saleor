Sets the customer as the owner of the checkout.

```graphql
checkoutCustomerAttach(
  checkoutId: ID!
  customerId: ID = 0
): CheckoutCustomerAttach

```

### Arguments

#### `checkoutId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the checkout.

#### `customerId` ([`ID`](/api-reference/scalars/id.mdx))

[Deprecated] The ID of the customer. To identify a customer you should authenticate with JWT. This field will be removed after 2020-07-31.

### Type

#### [`CheckoutCustomerAttach`](/api-reference/objects/checkout-customer-attach.mdx)

Sets the customer as the owner of the checkout.
