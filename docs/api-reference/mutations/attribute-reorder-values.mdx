Reorder the values of an attribute.

```graphql
attributeReorderValues(
  attributeId: ID!
  moves: [ReorderInput]!
): AttributeReorderValues

```

### Arguments

#### `attributeId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an attribute.

#### `moves` ([`[ReorderInput]!`](/api-reference/inputs/reorder-input.mdx))

The list of reordering operations for given attribute values.

### Type

#### [`AttributeReorderValues`](/api-reference/objects/attribute-reorder-values.mdx)

Reorder the values of an attribute.
