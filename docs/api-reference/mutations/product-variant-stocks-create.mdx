Creates stocks for product variant.

```graphql
productVariantStocksCreate(
  stocks: [StockInput!]!
  variantId: ID!
): ProductVariantStocksCreate

```

### Arguments

#### `stocks` ([`[StockInput!]!`](/api-reference/inputs/stock-input.mdx))

Input list of stocks to create.

#### `variantId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a product variant for which stocks will be created.

### Type

#### [`ProductVariantStocksCreate`](/api-reference/objects/product-variant-stocks-create.mdx)

Creates stocks for product variant.
