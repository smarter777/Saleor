---

Cancels existing fulfillment and optionally restocks items.

```graphql
orderFulfillmentCancel(
  id: ID!
  input: FulfillmentCancelInput!
): FulfillmentCancel

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an fulfillment to cancel.

#### `input` ([`FulfillmentCancelInput!`](/api-reference/inputs/fulfillment-cancel-input.mdx))

Fields required to cancel an fulfillment.

### Type

#### [`FulfillmentCancel`](/api-reference/objects/fulfillment-cancel.mdx)

Cancels existing fulfillment and optionally restocks items.
