Reorder the attributes of a page type.

```graphql
pageTypeReorderAttributes(
  moves: [ReorderInput!]!
  pageTypeId: ID!
): PageTypeReorderAttributes

```

### Arguments

#### `moves` ([`[ReorderInput!]!`](/api-reference/inputs/reorder-input.mdx))

The list of attribute reordering operations.

#### `pageTypeId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a page type.

### Type

#### [`PageTypeReorderAttributes`](/api-reference/objects/page-type-reorder-attributes.mdx)

Reorder the attributes of a page type.
