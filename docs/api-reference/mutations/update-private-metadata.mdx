Updates private metadata of an object.

```graphql
updatePrivateMetadata(
  id: ID!
  input: [MetadataInput!]!
): UpdatePrivateMetadata

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an object to update.

#### `input` ([`[MetadataInput!]!`](/api-reference/inputs/metadata-input.mdx))

Fields required to update the object's metadata.

### Type

#### [`UpdatePrivateMetadata`](/api-reference/objects/update-private-metadata.mdx)

Updates private metadata of an object.
