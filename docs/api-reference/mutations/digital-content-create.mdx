Create new digital content. This mutation must be sent as a `multipart` request. More detailed specs of the upload format can be found here: https://github.com/jaydenseric/graphql-multipart-request-spec

```graphql
digitalContentCreate(
  input: DigitalContentUploadInput!
  variantId: ID!
): DigitalContentCreate

```

### Arguments

#### `input` ([`DigitalContentUploadInput!`](/api-reference/inputs/digital-content-upload-input.mdx))

Fields required to create a digital content.

#### `variantId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a product variant to upload digital content.

### Type

#### [`DigitalContentCreate`](/api-reference/objects/digital-content-create.mdx)
