Capture an order.

```graphql
orderCapture(
  amount: PositiveDecimal!
  id: ID!
): OrderCapture

```

### Arguments

#### `amount` ([`PositiveDecimal!`](/api-reference/scalars/positive-decimal.mdx))

Amount of money to capture.

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the order to capture.

### Type

#### [`OrderCapture`](/api-reference/objects/order-capture.mdx)

Capture an order.
