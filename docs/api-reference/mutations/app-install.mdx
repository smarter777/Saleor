Install new app by using app manifest.

```graphql
appInstall(
  input: AppInstallInput!
): AppInstall

```

### Arguments

#### `input` ([`AppInstallInput!`](/api-reference/inputs/app-install-input.mdx))

Fields required to install a new app.

### Type

#### [`AppInstall`](/api-reference/objects/app-install.mdx)

Install new app by using app manifest.
