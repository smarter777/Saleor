Creates product variants for a given product.

```graphql
productVariantBulkCreate(
  product: ID!
  variants: [ProductVariantBulkCreateInput]!
): ProductVariantBulkCreate

```

### Arguments

#### `product` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the product to create the variants for.

#### `variants` ([`[ProductVariantBulkCreateInput]!`](/api-reference/inputs/product-variant-bulk-create-input.mdx))

Input list of product variants to create.

### Type

#### [`ProductVariantBulkCreate`](/api-reference/objects/product-variant-bulk-create.mdx)

Creates product variants for a given product.
