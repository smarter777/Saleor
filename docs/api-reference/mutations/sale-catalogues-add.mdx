Adds products, categories, collections to a voucher.

```graphql
saleCataloguesAdd(
  id: ID!
  input: CatalogueInput!
): SaleAddCatalogues

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a sale.

#### `input` ([`CatalogueInput!`](/api-reference/inputs/catalogue-input.mdx))

Fields required to modify catalogue IDs of sale.

### Type

#### [`SaleAddCatalogues`](/api-reference/objects/sale-add-catalogues.mdx)

Adds products, categories, collections to a voucher.
