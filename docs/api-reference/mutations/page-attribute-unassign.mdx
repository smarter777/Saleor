Unassign attributes from a given page type.

```graphql
pageAttributeUnassign(
  attributeIds: [ID!]!
  pageTypeId: ID!
): PageAttributeUnassign

```

### Arguments

#### `attributeIds` ([`[ID!]!`](/api-reference/scalars/id.mdx))

The IDs of the attributes to unassign.

#### `pageTypeId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the page type from which the attributes should be unassign.

### Type

#### [`PageAttributeUnassign`](/api-reference/objects/page-attribute-unassign.mdx)

Unassign attributes from a given page type.
