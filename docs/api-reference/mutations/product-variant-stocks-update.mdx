Update stocks for product variant.

```graphql
productVariantStocksUpdate(
  stocks: [StockInput!]!
  variantId: ID!

```

### Arguments

#### `stocks` ([`[StockInput!]!`](/api-reference/inputs/stock-input.mdx))

Input list of stocks to create.

#### `variantId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a product variant for which stocks will be created.

#### [`ProductVariantStocksUpdate`](/api-reference/objects/product-variant-stocks-update.mdx)
