Deactivate a gift card.

```graphql
giftCardDeactivate(
  id: ID!
): GiftCardDeactivate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a gift card to deactivate.

### Type

#### [`GiftCardDeactivate`](/api-reference/objects/gift-card-deactivate.mdx)

Deactivate a gift card.
