Changes ordering of the product image.

```graphql
productImageReorder(
  imagesIds: [ID]!
  productId: ID!
): ProductImageReorder

```

### Arguments

#### `imagesIds` ([`[ID]!`](/api-reference/scalars/id.mdx))

IDs of a product images in the desired order.

#### `productId` ([`ID!`](/api-reference/scalars/id.mdx))

Id of product that images order will be altered.

### Type

#### [`ProductImageReorder`](/api-reference/objects/product-image-reorder.mdx)

Changes ordering of the product image.
