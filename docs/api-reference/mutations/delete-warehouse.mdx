Deletes selected warehouse.

```graphql
deleteWarehouse(
  id: ID!
): WarehouseDelete

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a warehouse to delete.

### Type

#### [`WarehouseDelete`](/api-reference/objects/warehouse-delete.mdx)

Deletes selected warehouse.
