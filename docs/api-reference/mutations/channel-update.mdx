Update a channel.

```graphql
channelUpdate(
  id: ID!
  input: ChannelUpdateInput!
): ChannelUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a channel to update.

#### `input` ([`ChannelUpdateInput!`](/api-reference/inputs/channel-update-input.mdx))

Fields required to update a channel.

### Type

#### [`ChannelUpdate`](/api-reference/objects/channel-update.mdx)

Update a channel.
