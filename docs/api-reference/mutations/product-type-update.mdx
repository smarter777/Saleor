Updates an existing product type.

```graphql
productTypeUpdate(
  id: ID!
  input: ProductTypeInput!
): ProductTypeUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a product type to update.

#### `input` ([`ProductTypeInput!`](/api-reference/inputs/product-type-input.mdx))

Fields required to update a product type.

### Type

#### [`ProductTypeUpdate`](/api-reference/objects/product-type-update.mdx)

Updates an existing product type.
