Sets a default address for the given user.

```graphql
addressSetDefault(
  addressId: ID!
  type: AddressTypeEnum!
  userId: ID!
): AddressSetDefault

```

### Arguments

#### `addressId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the address.

#### `type` ([`AddressTypeEnum!`](/api-reference/enums/address-type-enum.mdx))

The type of address.

#### `userId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the user to change the address for.

### Type

#### [`AddressSetDefault`](/api-reference/objects/address-set-default.mdx)

Sets a default address for the given user.
