Requests deletion of an invoice.

```graphql
invoiceRequestDelete(
  id: ID!
): InvoiceRequestDelete

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an invoice to request the deletion.

### Type

#### [`InvoiceRequestDelete`](/api-reference/objects/invoice-request-delete.mdx)

Requests deletion of an invoice.
