Set default variant for a product. Mutation triggers PRODUCT_UPDATED webhook.

```graphql
productVariantSetDefault(
  productId: ID!
  variantId: ID!
): ProductVariantSetDefault

```

### Arguments

#### `productId` ([`ID!`](/api-reference/scalars/id.mdx))

Id of a product that will have the default variant set.

#### `variantId` ([`ID!`](/api-reference/scalars/id.mdx))

Id of a variant that will be set as default.

### Type

#### [`ProductVariantSetDefault`](/api-reference/objects/product-variant-set-default.mdx)

Set default variant for a product. Mutation triggers PRODUCT_UPDATED webhook.
