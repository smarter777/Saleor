Retry failed installation of new app.

```graphql
appRetryInstall(
  activateAfterInstallation: Boolean = "true"
  id: ID!
): AppRetryInstall

```

### Arguments

#### `activateAfterInstallation` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Determine if app will be set active or not.

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of failed installation.

### Type

#### [`AppRetryInstall`](/api-reference/objects/app-retry-install.mdx)

Retry failed installation of new app.
