Exclude products from shipping price.

```graphql
shippingPriceExcludeProducts(
  id: ID!
  input: ShippingPriceExcludeProductsInput!
): ShippingPriceExcludeProducts

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a shipping price.

#### `input` ([`ShippingPriceExcludeProductsInput!`](/api-reference/inputs/shipping-price-exclude-products-input.mdx))

Exclude products input.

### Type

#### [`ShippingPriceExcludeProducts`](/api-reference/objects/shipping-price-exclude-products.mdx)

Exclude products from shipping price.
