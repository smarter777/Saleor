Updates an existing product.

```graphql
productUpdate(
  id: ID!
  input: ProductInput!
): ProductUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a product to update.

#### `input` ([`ProductInput!`](/api-reference/inputs/product-input.mdx))

Fields required to update a product.

### Type

#### [`ProductUpdate`](/api-reference/objects/product-update.mdx)

Updates an existing product.
