Adds a checkout line to the existing checkout.

```graphql
checkoutLinesAdd(
  checkoutId: ID!
  lines: [CheckoutLineInput]!
): CheckoutLinesAdd

```

### Arguments

#### `checkoutId` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the checkout.

#### `lines` ([`[CheckoutLineInput]!`](/api-reference/inputs/checkout-line-input.mdx))

A list of checkout lines, each containing information about an item in the checkout.

### Type

#### [`CheckoutLinesAdd`](/api-reference/objects/checkout-lines-add.mdx)

Adds a checkout line to the existing checkout.
