Confirms an unconfirmed order by changing status to unfulfilled.

```graphql
orderConfirm(
  id: ID!
): OrderConfirm

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an order to confirm.

### Type

#### [`OrderConfirm`](/api-reference/objects/order-confirm.mdx)

Confirms an unconfirmed order by changing status to unfulfilled.
