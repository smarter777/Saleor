Sends an email with the account removal link for the logged-in user.

```graphql
accountRequestDeletion(
  redirectUrl: String!
): AccountRequestDeletion

```

### Arguments

#### `redirectUrl` ([`String!`](/api-reference/scalars/string.mdx))

URL of a view where users should be redirected to delete their account. URL in RFC 1808 format.

### Type

#### [`AccountRequestDeletion`](/api-reference/objects/account-request-deletion.mdx)

Sends an email with the account removal link for the logged-in user.
