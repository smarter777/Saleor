title: productVariantReorder

Reorder the variants of a product. Mutation updates updated_at on product and triggers PRODUCT_UPDATED webhook.

```graphql
productVariantReorder(
  moves: [ReorderInput]!
  productId: ID!
): ProductVariantReorder

```

### Arguments

#### `moves` ([`[ReorderInput]!`](/api-reference/inputs/reorder-input.mdx))

The list of variant reordering operations.

#### `productId` ([`ID!`](/api-reference/scalars/id.mdx))

Id of product that variants order will be altered.

### Type

#### [`ProductVariantReorder`](/api-reference/objects/product-variant-reorder.mdx)

Reorder the variants of a product. Mutation updates updated_at on product and triggers PRODUCT_UPDATED webhook.
