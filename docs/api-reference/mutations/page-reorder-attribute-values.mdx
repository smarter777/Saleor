Reorder page attribute values.

```graphql
pageReorderAttributeValues(
  attributeId: ID!
  moves: [ReorderInput]!
  pageId: ID!
): PageReorderAttributeValues

```

### Arguments

#### `attributeId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of an attribute.

#### `moves` ([`[ReorderInput]!`](/api-reference/inputs/reorder-input.mdx))

The list of reordering operations for given attribute values.

#### `pageId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a page.

### Type

#### [`PageReorderAttributeValues`](/api-reference/objects/page-reorder-attribute-values.mdx)

Reorder page attribute values.
