Unassign an image from a product variant.

```graphql
variantImageUnassign(
  imageId: ID!
  variantId: ID!
): VariantImageUnassign

```

### Arguments

#### `imageId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a product image to unassign from a variant.

#### `variantId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a product variant.

### Type

#### [`VariantImageUnassign`](/api-reference/objects/variant-image-unassign.mdx)

Unassign an image from a product variant.
