Updates an address of the logged-in user.

```graphql
accountAddressUpdate(
  id: ID!
  input: AddressInput!
): AccountAddressUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the address to update.

#### `input` ([`AddressInput!`](/api-reference/inputs/address-input.mdx))

Fields required to update the address.

### Type

#### [`AccountAddressUpdate`](/api-reference/objects/account-address-update.mdx)

Updates an address of the logged-in user.
