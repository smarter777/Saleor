Updates a category.

```graphql
categoryUpdate(
  id: ID!
  input: CategoryInput!
): CategoryUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a category to update.

#### `input` ([`CategoryInput!`](/api-reference/inputs/category-input.mdx))

Fields required to update a category.

### Type

#### [`CategoryUpdate`](/api-reference/objects/category-update.mdx)

Updates a category.
