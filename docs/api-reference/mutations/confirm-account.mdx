Confirm user account with token sent by email during registration.

```graphql
confirmAccount(
  email: String!
  token: String!
): ConfirmAccount

```

### Arguments

#### `email` ([`String!`](/api-reference/scalars/string.mdx))

E-mail of the user performing account confirmation.

#### `token` ([`String!`](/api-reference/scalars/string.mdx))

A one-time token required to confirm the account.

### Type

#### [`ConfirmAccount`](/api-reference/objects/confirm-account.mdx)

Confirm user account with token sent by email during registration.
