Moves items of menus.

```graphql
menuItemMove(
  menu: ID!
  moves: [MenuItemMoveInput]!
): MenuItemMove

```

### Arguments

#### `menu` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the menu.

#### `moves` ([`[MenuItemMoveInput]!`](/api-reference/inputs/menu-item-move-input.mdx))

The menu position data.

### Type

#### [`MenuItemMove`](/api-reference/objects/menu-item-move.mdx)

Moves items of menus.
