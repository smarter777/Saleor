Assigns storefront's navigation menus.

```graphql
assignNavigation(
  menu: ID = 0
  navigationType: NavigationType!
): AssignNavigation

```

### Arguments

#### `menu` ([`ID`](/api-reference/scalars/id.mdx))

ID of the menu.

#### `navigationType` ([`NavigationType!`](/api-reference/enums/navigation-type.mdx))

Type of the navigation bar to assign the menu to.

### Type

#### [`AssignNavigation`](/api-reference/objects/assign-navigation.mdx)

Assigns storefront's navigation menus.
