Remove user account.

```graphql
accountDelete(
  token: String!
): AccountDelete

```

### Arguments

#### `token` ([`String!`](/api-reference/scalars/string.mdx))

A one-time token required to remove account. Sent by email using AccountRequestDeletion mutation.

### Type

#### [`AccountDelete`](/api-reference/objects/account-delete.mdx)

Remove user account.
