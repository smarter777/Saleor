Mark order as manually paid.

```graphql
orderMarkAsPaid(
  id: ID!
  transactionReference: String
): OrderMarkAsPaid

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the order to mark paid.

#### `transactionReference` ([`String`](/api-reference/scalars/string.mdx))

The external transaction reference.

### Type

#### [`OrderMarkAsPaid`](/api-reference/objects/order-mark-as-paid.mdx)

Mark order as manually paid.
