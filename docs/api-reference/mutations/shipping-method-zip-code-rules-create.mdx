Create a new zip code exclusion range for shipping method.

```graphql
shippingMethodZipCodeRulesCreate(
  input: ShippingZipCodeRulesCreateInput!
  shippingMethodId: ID!
): ShippingZipCodeRulesCreate

```

### Arguments

#### `input` ([`ShippingZipCodeRulesCreateInput!`](/api-reference/inputs/shipping-zip-code-rules-create-input.mdx))

Fields required to create a shipping zip codes.

#### `shippingMethodId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a shipping method to assign.

### Type

#### [`ShippingZipCodeRulesCreate`](/api-reference/objects/shipping-zip-code-rules-create.mdx)

Create a new zip code exclusion range for shipping method.
