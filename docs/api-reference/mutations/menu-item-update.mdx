Updates a menu item.

```graphql
menuItemUpdate(
  id: ID!
  input: MenuItemInput!
): MenuItemUpdate

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a menu item to update.

#### `input` ([`MenuItemInput!`](/api-reference/inputs/menu-item-input.mdx))

Fields required to update a menu item. Only one of `url`, `category`, `page`, `collection` is allowed per item.

### Type

#### [`MenuItemUpdate`](/api-reference/objects/menu-item-update.mdx)

Updates a menu item.
