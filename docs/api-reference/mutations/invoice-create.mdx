Creates a ready to send invoice.

```graphql
invoiceCreate(
  input: InvoiceCreateInput!
  orderId: ID!
): InvoiceCreate

```

### Arguments

#### `input` ([`InvoiceCreateInput!`](/api-reference/inputs/invoice-create-input.mdx))

Fields required when creating an invoice.

#### `orderId` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the order related to invoice.

### Type

#### [`InvoiceCreate`](/api-reference/objects/invoice-create.mdx)

Creates a ready to send invoice.
