Change the password of the logged in user.

```graphql
passwordChange(
  newPassword: String!
  oldPassword: String!
): PasswordChange

```

### Arguments

#### `newPassword` ([`String!`](/api-reference/scalars/string.mdx))

New user password.

#### `oldPassword` ([`String!`](/api-reference/scalars/string.mdx))

Current user password.

### Type

#### [`PasswordChange`](/api-reference/objects/password-change.mdx)

Change the password of the logged in user.
