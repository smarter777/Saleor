Remove shipping zone from given warehouse.

```graphql
unassignWarehouseShippingZone(
  id: ID!
  shippingZoneIds: [ID!]!
): WarehouseShippingZoneUnassign

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of a warehouse to update.

#### `shippingZoneIds` ([`[ID!]!`](/api-reference/scalars/id.mdx))

List of shipping zone IDs.

### Type

#### [`WarehouseShippingZoneUnassign`](/api-reference/objects/warehouse-shipping-zone-unassign.mdx)

Remove shipping zone from given warehouse.
