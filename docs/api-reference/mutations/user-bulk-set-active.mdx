Activate or deactivate users.

```graphql
userBulkSetActive(
  ids: [ID]!
  isActive: Boolean!
): UserBulkSetActive

```

### Arguments

#### `ids` ([`[ID]!`](/api-reference/scalars/id.mdx))

List of user IDs to (de)activate).

#### `isActive` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Determine if users will be set active or not.

### Type

#### [`UserBulkSetActive`](/api-reference/objects/user-bulk-set-active.mdx)

Activate or deactivate users.
