Adds note to the order.

```graphql
orderAddNote(
  order: ID!
  input: OrderAddNoteInput!
): OrderAddNote

```

### Arguments

#### `order` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the order to add a note for.

#### `input` ([`OrderAddNoteInput!`](/api-reference/inputs/order-add-note-input.mdx))

Fields required to create a note for the order.

### Type

#### [`OrderAddNote`](/api-reference/objects/order-add-note.mdx)

Adds note to the order.
