type PageInput {

}

### Fields

#### `slug` ([`String`](/api-reference/scalars/string.mdx))

Page internal name.

#### `title` ([`String`](/api-reference/scalars/string.mdx))

Page title.

#### `content` ([`String`](/api-reference/scalars/string.mdx))

Page content. May consist of ordinary text, HTML and images.

#### `contentJson` ([`JSONString`](/api-reference/scalars/json-string.mdx))

Page content in JSON format.

#### `attributes` ([`[AttributeValueInput!]`](/api-reference/inputs/attribute-value-input.mdx))

List of attributes.

#### `isPublished` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Determines if page is visible in the storefront.

#### `publicationDate` ([`String`](/api-reference/scalars/string.mdx))

Publication date. ISO 8601 standard.

#### `seo` ([`SeoInput`](/api-reference/inputs/seo-input.mdx))

Search engine optimization fields.
