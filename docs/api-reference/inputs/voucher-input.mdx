```graphql
type VoucherInput {
  type: VoucherTypeEnum
  name: String
  code: String
  startDate: DateTime
  endDate: DateTime
  discountValueType: DiscountValueTypeEnum
  products: [ID]
  collections: [ID]
  categories: [ID]
  minCheckoutItemsQuantity: Int
  countries: [String]
  applyOncePerOrder: Boolean
  applyOncePerCustomer: Boolean
  usageLimit: Int
}
```

### Fields

#### `type` ([`VoucherTypeEnum`](/api-reference/enums/voucher-type-enum.mdx))

Voucher type: PRODUCT, CATEGORY SHIPPING or ENTIRE_ORDER.

#### `name` ([`String`](/api-reference/scalars/string.mdx))

Voucher name.

#### `code` ([`String`](/api-reference/scalars/string.mdx))

Code to use the voucher.

#### `startDate` ([`DateTime`](/api-reference/scalars/date-time.mdx))

Start date of the voucher in ISO 8601 format.

#### `endDate` ([`DateTime`](/api-reference/scalars/date-time.mdx))

End date of the voucher in ISO 8601 format.

#### `discountValueType` ([`DiscountValueTypeEnum`](/api-reference/enums/discount-value-type-enum.mdx))

Choices: fixed or percentage.

#### `products` ([`[ID]`](/api-reference/scalars/id.mdx))

Products discounted by the voucher.

#### `collections` ([`[ID]`](/api-reference/scalars/id.mdx))

Collections discounted by the voucher.

#### `categories` ([`[ID]`](/api-reference/scalars/id.mdx))

Categories discounted by the voucher.

#### `minCheckoutItemsQuantity` ([`Int`](/api-reference/scalars/int.mdx))

Minimal quantity of checkout items required to apply the voucher.

#### `countries` ([`[String]`](/api-reference/scalars/string.mdx))

Country codes that can be used with the shipping voucher.

#### `applyOncePerOrder` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Voucher should be applied to the cheapest item or entire order.

#### `applyOncePerCustomer` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Voucher should be applied once per customer.

#### `usageLimit` ([`Int`](/api-reference/scalars/int.mdx))

Limit number of times this voucher can be used in total.
