```graphql
type AttributeValueInput {
  id: ID
  values: [String]

}
```

### Fields

#### `id` ([`ID`](/api-reference/scalars/id.mdx))

ID of the selected attribute.

#### `values` ([`[String]`](/api-reference/scalars/string.mdx))

The value or slug of an attribute to resolve. If the passed value is non-existent, it will be created.

#### `file` ([`String`](/api-reference/scalars/string.mdx))

URL of the file attribute. Every time, a new value is created.

#### `contentType` ([`String`](/api-reference/scalars/string.mdx))

File content type.
