Represents a shop resource containing general shop data and configuration.

```graphql
type Shop {
  availablePaymentGateways(currency: String): [PaymentGateway!]!
  availableShippingMethods(
    channel: String!
    address: AddressInput
  ): [ShippingMethod]
  geolocalization: Geolocalization
  authorizationKeys: [AuthorizationKey]!
  countries(languageCode: LanguageCodeEnum): [CountryDisplay!]!
  defaultCountry: CountryDisplay
  defaultMailSenderName: String
  defaultMailSenderAddress: String
  description: String
  domain: Domain!
  languages: [LanguageDisplay]!
  name: String!
  navigation: Navigation
  permissions: [Permission]!
  phonePrefixes: [String]!
  headerText: String
  includeTaxesInPrices: Boolean!
  displayGrossPrices: Boolean!
  chargeTaxesOnShipping: Boolean!
  trackInventoryByDefault: Boolean
  defaultWeightUnit: WeightUnitsEnum
  translation(languageCode: LanguageCodeEnum!): ShopTranslation
  automaticFulfillmentDigitalProducts: Boolean
  defaultDigitalMaxDownloads: Int
  defaultDigitalUrlValidDays: Int
  companyAddress: Address
  customerSetPasswordUrl: String
  staffNotificationRecipients: [StaffNotificationRecipient]
}
```

### Fields

#### `availablePaymentGateways` ([`[PaymentGateway!]!`](/api-reference/objects/payment-gateway.mdx))

List of available payment gateways.

- ##### `currency` ([`String`](/api-reference/scalars/string.mdx))

A currency for which gateways will be returned.

#### `availableShippingMethods` ([`[ShippingMethod]`](/api-reference/objects/shipping-method.mdx))

Shipping methods that are available for the shop.

- ##### `channel` ([`String!`](/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

- ##### `address` ([`AddressInput`](/api-reference/inputs/address-input.mdx))

Address for which available shipping methods should be returned.

#### `geolocalization` ([`Geolocalization`](/api-reference/objects/geolocalization.mdx))

Customer's geolocalization data.

#### `authorizationKeys` ([`[AuthorizationKey]!`](/api-reference/objects/authorization-key.mdx))

List of configured authorization keys. Authorization keys are used to enable third-party OAuth authorization (currently Facebook or Google).

#### `countries` ([`[CountryDisplay!]!`](/api-reference/objects/country-display.mdx))

List of countries available in the shop.

- ##### `languageCode` ([`LanguageCodeEnum`](/api-reference/enums/language-code-enum.mdx))

A language code to return the translation for.

#### `defaultCountry` ([`CountryDisplay`](/api-reference/objects/country-display.mdx))

Shop's default country.

#### `defaultMailSenderName` ([`String`](/api-reference/scalars/string.mdx))

Default shop's email sender's name.

#### `defaultMailSenderAddress` ([`String`](/api-reference/scalars/string.mdx))

Default shop's email sender's address.

#### `description` ([`String`](/api-reference/scalars/string.mdx))

Shop's description.

#### `domain` ([`Domain!`](/api-reference/objects/domain.mdx))

Shop's domain data.

#### `languages` ([`[LanguageDisplay]!`](/api-reference/objects/language-display.mdx))

List of the shops's supported languages.

#### `name` ([`String!`](/api-reference/scalars/string.mdx))

Shop's name.

#### `navigation` ([`Navigation`](/api-reference/objects/navigation.mdx))

<sub><sup><Tag color="#ffba00">DEPRECATED</Tag> Fetch menus using the `menu` query with `slug` parameter.</sup></sub>

Shop's navigation.

#### `permissions` ([`[Permission]!`](/api-reference/objects/permission.mdx))

List of available permissions.

#### `phonePrefixes` ([`[String]!`](/api-reference/scalars/string.mdx))

List of possible phone prefixes.

#### `headerText` ([`String`](/api-reference/scalars/string.mdx))

Header text.

#### `includeTaxesInPrices` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Include taxes in prices.

#### `displayGrossPrices` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Display prices with tax in store.

#### `chargeTaxesOnShipping` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Charge taxes on shipping.

#### `trackInventoryByDefault` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Enable inventory tracking.

#### `defaultWeightUnit` ([`WeightUnitsEnum`](/api-reference/enums/weight-units-enum.mdx))

Default weight unit.

#### `translation` ([`ShopTranslation`](/api-reference/objects/shop-translation.mdx))

Returns translated shop fields for the given language code.

- ##### `languageCode` ([`LanguageCodeEnum!`](/api-reference/enums/language-code-enum.mdx))

A language code to return the translation for shop.

#### `automaticFulfillmentDigitalProducts` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Enable automatic fulfillment for all digital products.

#### `defaultDigitalMaxDownloads` ([`Int`](/api-reference/scalars/int.mdx))

Default number of max downloads per digital content URL.

#### `defaultDigitalUrlValidDays` ([`Int`](/api-reference/scalars/int.mdx))

Default number of days which digital content URL will be valid.

#### `companyAddress` ([`Address`](/api-reference/objects/address.mdx))

Company address.

#### `customerSetPasswordUrl` ([`String`](/api-reference/scalars/string.mdx))

URL of a view where customers can set their password.

#### `staffNotificationRecipients` ([`[StaffNotificationRecipient]`](/api-reference/objects/staff-notification-recipient.mdx))

List of staff notification recipients.
