Represents a job data of exported file.

```graphql
type ExportFile implements Node, Job {
  id: ID!
  user: User
  app: App
  status: JobStatusEnum!
  createdAt: DateTime!
  updatedAt: DateTime!
  message: String
  url: String
  events: [ExportEvent!]
}
```

### Fields

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `user` ([`User`](/api-reference/objects/user.mdx))

#### `app` ([`App`](/api-reference/objects/app.mdx))

#### `status` ([`JobStatusEnum!`](/api-reference/enums/job-status-enum.mdx))

Job status.

#### `createdAt` ([`DateTime!`](/api-reference/scalars/date-time.mdx))

Created date time of job in ISO 8601 format.

#### `updatedAt` ([`DateTime!`](/api-reference/scalars/date-time.mdx))

Date time of job last update in ISO 8601 format.

#### `message` ([`String`](/api-reference/scalars/string.mdx))

Job message.

#### `url` ([`String`](/api-reference/scalars/string.mdx))

The URL of field to download.

#### `events` ([`[ExportEvent!]`](/api-reference/objects/export-event.mdx))

List of events associated with the export.
