Represents a monetary value with taxes. In cases where taxes were not applied, net and gross values will be equal.

```graphql
type TaxedMoney {
  currency: String!
  gross: Money!
  net: Money!
  tax: Money!
}
```

### Fields

#### `currency` ([`String!`](/api-reference/scalars/string.mdx))

Currency code.

#### `gross` ([`Money!`](/api-reference/objects/money.mdx))

Amount of money including taxes.

#### `net` ([`Money!`](/api-reference/objects/money.mdx))

Amount of money without taxes.

#### `tax` ([`Money!`](/api-reference/objects/money.mdx))

Amount of taxes.
