Represents an Invoice.

```graphql
type Invoice implements ObjectWithMetadata, Job, Node {
  id: ID!
  metadata: [MetadataItem]!
  status: JobStatusEnum!
  number: String
  externalUrl: String
  privateMetadata: [MetadataItem]!
  createdAt: DateTime!
  updatedAt: DateTime!
  message: String
  url: String
}
```

### Fields

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `metadata` ([`[MetadataItem]!`](/api-reference/objects/metadata-item.mdx))

List of public metadata items. Can be accessed without permissions.

#### `status` ([`JobStatusEnum!`](/api-reference/enums/job-status-enum.mdx))

Job status.

#### `number` ([`String`](/api-reference/scalars/string.mdx))

#### `externalUrl` ([`String`](/api-reference/scalars/string.mdx))

#### `privateMetadata` ([`[MetadataItem]!`](/api-reference/objects/metadata-item.mdx))

List of private metadata items.Requires proper staff permissions to access.

#### `createdAt` ([`DateTime!`](/api-reference/scalars/date-time.mdx))

Created date time of job in ISO 8601 format.

#### `updatedAt` ([`DateTime!`](/api-reference/scalars/date-time.mdx))

Date time of job last update in ISO 8601 format.

#### `message` ([`String`](/api-reference/scalars/string.mdx))

Job message.

#### `url` ([`String`](/api-reference/scalars/string.mdx))

URL to download an invoice.
