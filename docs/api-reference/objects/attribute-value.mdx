Represents a value of an attribute.

```graphql
type AttributeValue implements Node {
  id: ID!
  name: String
  slug: String
  type: AttributeValueType
  translation(languageCode: LanguageCodeEnum!): AttributeValueTranslation
  inputType: AttributeInputTypeEnum
  file: File
}
```

### Fields

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `name` ([`String`](/api-reference/scalars/string.mdx))

Name of a value displayed in the interface.

#### `slug` ([`String`](/api-reference/scalars/string.mdx))

Internal representation of a value (unique per attribute).

#### `type` ([`AttributeValueType`](/api-reference/enums/attribute-value-type.mdx))

<sub><sup><Tag color="#ffba00">DEPRECATED</Tag> Use the `inputType` field to determine the type of attribute's value. This field will be removed after 2020-07-31.</sup></sub>

Type of value (used only when `value` field is set).

#### `translation` ([`AttributeValueTranslation`](/api-reference/objects/attribute-value-translation.mdx))

Returns translated attribute value fields for the given language code.

- ##### `languageCode` ([`LanguageCodeEnum!`](/api-reference/enums/language-code-enum.mdx))

A language code to return the translation for attribute value.

#### `inputType` ([`AttributeInputTypeEnum`](/api-reference/enums/attribute-input-type-enum.mdx))

The input type to use for entering attribute values in the dashboard.

#### `file` ([`File`](/api-reference/objects/file.mdx))

Represents file URL and content type (if attribute value is a file).

### Interfaces

#### [`Node`](/api-reference/interfaces/node.mdx)

An object with an ID
