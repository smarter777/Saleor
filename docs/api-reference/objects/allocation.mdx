Represents allocation.

```graphql
type Allocation implements Node {
  id: ID!
  quantity: Int!

```

### Fields

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `quantity` ([`Int!`](/api-reference/scalars/int.mdx))

Quantity allocated for orders.

#### `warehouse` ([`Warehouse!`](/api-reference/objects/warehouse.mdx))

The warehouse were items were allocated.

### Interfaces
