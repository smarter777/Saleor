Event sent when app is updated.

Added in Saleor 3.4.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

```graphql
type AppUpdated implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  app: App
}
```
