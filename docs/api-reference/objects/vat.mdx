Represents a VAT rate for a country.

```graphql
type VAT {
  countryCode: String!
  standardRate: Float
  reducedRates: [ReducedRate]!
}
```

### Fields

#### `countryCode` ([`String!`](/api-reference/scalars/string.mdx))

Country code.

#### `standardRate` ([`Float`](/api-reference/scalars/float.mdx))

Standard VAT rate in percent.

#### `reducedRates` ([`[ReducedRate]!`](/api-reference/objects/reduced-rate.mdx))

Country's VAT rate exceptions for specific types of goods.
