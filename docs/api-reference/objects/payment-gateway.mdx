```graphql
  name: String!
  id: ID!

  currencies: [String]!
}
```

#### `config` ([`[GatewayConfigLine!]!`](/api-reference/objects/gateway-config-line.mdx))

#### `currencies` ([`[String]!`](/api-reference/scalars/string.mdx))
