title: PaymentGateway
---

Available payment gateway backend with configuration necessary to setup client.

```graphql
type PaymentGateway {
  name: String!
  id: ID!
  config: [GatewayConfigLine!]!
  currencies: [String]!
}
```

### Fields

#### `name` ([`String!`](/api-reference/scalars/string.mdx))

Payment gateway name.

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

Payment gateway ID.

#### `config` ([`[GatewayConfigLine!]!`](/api-reference/objects/gateway-config-line.mdx))

Payment gateway client configuration.

#### `currencies` ([`[String]!`](/api-reference/scalars/string.mdx))

Payment gateway supported currencies.
