### Fields

#### `field` ([`String`](/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`ShippingErrorCode!`](/api-reference/enums/shipping-error-code.mdx))

The error code.

#### `warehouses` ([`[ID!]`](/api-reference/scalars/id.mdx))

List of warehouse IDs which causes the error.

#### `channels` ([`[ID!]`](/api-reference/scalars/id.mdx))

List of channels IDs which causes the error.
