Represents app data.

```graphql
type App implements Node, ObjectWithMetadata {
  id: ID!
  name: String
  created: DateTime
  isActive: Boolean
  permissions: [Permission]
  tokens: [AppToken]
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  type: AppTypeEnum
  webhooks: [Webhook]
  aboutApp: String
  dataPrivacy: String
  dataPrivacyUrl: String
  homepageUrl: String
  supportUrl: String
  configurationUrl: String
  appUrl: String
  version: String
  accessToken: String
}
```

### Fields

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `name` ([`String`](/api-reference/scalars/string.mdx))

Name of the app.

#### `created` ([`DateTime`](/api-reference/scalars/date-time.mdx))

The date and time when the app was created.

#### `isActive` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Determine if app will be set active or not.

#### `permissions` ([`[Permission]`](/api-reference/objects/permission.mdx))

List of the app's permissions.

#### `tokens` ([`[AppToken]`](/api-reference/objects/app-token.mdx))

Last 4 characters of the tokens.

#### `privateMetadata` ([`[MetadataItem]!`](/api-reference/objects/metadata-item.mdx))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](/api-reference/objects/metadata-item.mdx))

List of public metadata items. Can be accessed without permissions.

#### `type` ([`AppTypeEnum`](/api-reference/enums/app-type-enum.mdx))

Type of the app.

#### `webhooks` ([`[Webhook]`](/api-reference/objects/webhook.mdx))

List of webhooks assigned to this app.

#### `aboutApp` ([`String`](/api-reference/scalars/string.mdx))

Description of this app.

#### `dataPrivacy` ([`String`](/api-reference/scalars/string.mdx))

Description of the data privacy defined for this app.

#### `dataPrivacyUrl` ([`String`](/api-reference/scalars/string.mdx))

Url to details about the privacy policy on the app owner page.

#### `homepageUrl` ([`String`](/api-reference/scalars/string.mdx))

Homepage of the app.

#### `supportUrl` ([`String`](/api-reference/scalars/string.mdx))

Support page for the app.

#### `configurationUrl` ([`String`](/api-reference/scalars/string.mdx))

Url to iframe with the configuration for the app.

#### `appUrl` ([`String`](/api-reference/scalars/string.mdx))

Url to iframe with the app.

#### `version` ([`String`](/api-reference/scalars/string.mdx))

Version number of the app.

#### `accessToken` ([`String`](/api-reference/scalars/string.mdx))

JWT token used to authenticate by thridparty app.
