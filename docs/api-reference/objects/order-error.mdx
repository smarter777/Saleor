No description

```graphql
type OrderError {
  field: String
  message: String
  code: OrderErrorCode!
  warehouse: ID
  orderLine: ID
  variants: [ID!]
}
```

### Fields

#### `field` ([`String`](/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`OrderErrorCode!`](/api-reference/enums/order-error-code.mdx))

The error code.

#### `warehouse` ([`ID`](/api-reference/scalars/id.mdx))

Warehouse ID which causes the error.

#### `orderLine` ([`ID`](/api-reference/scalars/id.mdx))

Order line ID which causes the error.

#### `variants` ([`[ID!]`](/api-reference/scalars/id.mdx))

List of product variants that are associated with the error
