title: ProductVariant

Represents a version of a product such as different size or color.

```graphql
type ProductVariant implements Node, ObjectWithMetadata {
  id: ID!
  name: String!
  sku: String!
  product: Product!
  trackInventory: Boolean!
  weight: Weight
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  channelListings: [ProductVariantChannelListing!]
  pricing: VariantPricingInfo
  attributes(variantSelection: VariantAttributeScope): [SelectedAttribute!]!
  costPrice: Money
  margin: Int
  quantityOrdered: Int
  revenue(period: ReportingPeriod): TaxedMoney
  images: [ProductImage]
  translation(languageCode: LanguageCodeEnum!): ProductVariantTranslation
  digitalContent: DigitalContent
  stocks(countryCode: CountryCode): [Stock]
  quantityAvailable(countryCode: CountryCode): Int!
}
```

### Fields

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `name` ([`String!`](/api-reference/scalars/string.mdx))

#### `sku` ([`String!`](/api-reference/scalars/string.mdx))

#### `product` ([`Product!`](/api-reference/objects/product.mdx))

#### `trackInventory` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

#### `weight` ([`Weight`](/api-reference/objects/weight.mdx))

#### `privateMetadata` ([`[MetadataItem]!`](/api-reference/objects/metadata-item.mdx))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](/api-reference/objects/metadata-item.mdx))

List of public metadata items. Can be accessed without permissions.

#### `channelListings` ([`[ProductVariantChannelListing!]`](/api-reference/objects/product-variant-channel-listing.mdx))

List of price information in channels for the product.

#### `pricing` ([`VariantPricingInfo`](/api-reference/objects/variant-pricing-info.mdx))

Lists the storefront variant's pricing, the current price and discounts, only meant for displaying.

#### `attributes` ([`[SelectedAttribute!]!`](/api-reference/objects/selected-attribute.mdx))

List of attributes assigned to this variant.

- ##### `variantSelection` ([`VariantAttributeScope`](/api-reference/enums/variant-attribute-scope.mdx))

Define scope of returned attributes.

#### `costPrice` ([`Money`](/api-reference/objects/money.mdx))

Cost price of the variant.

#### `margin` ([`Int`](/api-reference/scalars/int.mdx))

Gross margin percentage value.

#### `quantityOrdered` ([`Int`](/api-reference/scalars/int.mdx))

Total quantity ordered.

#### `revenue` ([`TaxedMoney`](/api-reference/objects/taxed-money.mdx))

Total revenue generated by a variant in given period of time. Note: this field should be queried using `reportProductSales` query as it uses optimizations suitable for such calculations.

- ##### `period` ([`ReportingPeriod`](/api-reference/enums/reporting-period.mdx))

#### `images` ([`[ProductImage]`](/api-reference/objects/product-image.mdx))

List of images for the product variant.

#### `translation` ([`ProductVariantTranslation`](/api-reference/objects/product-variant-translation.mdx))

Returns translated product variant fields for the given language code.

- ##### `languageCode` ([`LanguageCodeEnum!`](/api-reference/enums/language-code-enum.mdx))

A language code to return the translation for product variant.

#### `digitalContent` ([`DigitalContent`](/api-reference/objects/digital-content.mdx))

Digital content for the product variant.

#### `stocks` ([`[Stock]`](/api-reference/objects/stock.mdx))

Stocks for the product variant.

- ##### `countryCode` ([`CountryCode`](/api-reference/enums/country-code.mdx))

Two-letter ISO 3166-1 country code.

#### `quantityAvailable` ([`Int!`](/api-reference/scalars/int.mdx))

Quantity of a product available for sale in one checkout.

- ##### `countryCode` ([`CountryCode`](/api-reference/enums/country-code.mdx))

Two-letter ISO 3166-1 country code. When provided, the exact quantity from a warehouse operating in shipping zones that contain this country will be returned. Otherwise, it will return the maximum quantity from all shipping zones.
