Represents a range of amounts of money.

```graphql
type MoneyRange {
  start: Money
  stop: Money
}
```

### Fields

#### `start` ([`Money`](/api-reference/objects/money.mdx))

Lower bound of a price range.

#### `stop` ([`Money`](/api-reference/objects/money.mdx))

Upper bound of a price range.
