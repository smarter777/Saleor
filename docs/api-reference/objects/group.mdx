Represents permission group data.

```graphql
type Group implements Node {
  id: ID!
  name: String!
  permissions: [Permission]
  users: [User]
  userCanManage: Boolean!
}
```

### Fields

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `name` ([`String!`](/api-reference/scalars/string.mdx))

#### `permissions` ([`[Permission]`](/api-reference/objects/permission.mdx))

List of group permissions

#### `users` ([`[User]`](/api-reference/objects/user.mdx))

List of group users

#### `userCanManage` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

True, if the currently authenticated user has rights to manage a group.

### Interfaces

#### [`Node`](/api-reference/interfaces/node.mdx)

An object with an ID
