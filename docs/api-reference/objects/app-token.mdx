Represents token data.

```graphql
type AppToken implements Node {
  name: String
  authToken: String
  id: ID!
}
```

### Fields

#### `name` ([`String`](/api-reference/scalars/string.mdx))

Name of the authenticated token.

#### `authToken` ([`String`](/api-reference/scalars/string.mdx))

Last 4 characters of the token.

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

### Interfaces

#### [`Node`](/api-reference/interfaces/node.mdx)

An object with an ID
