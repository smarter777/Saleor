```graphql
type WebhookError {
  field: String
  message: String
  code: WebhookErrorCode!
```

### Fields

#### `field` ([`String`](/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`WebhookErrorCode!`](/api-reference/enums/webhook-error-code.mdx))

The error code.
