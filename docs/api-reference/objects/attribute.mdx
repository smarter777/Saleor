Custom attribute of a product. Attributes can be assigned to products and variants at the product type level.

```graphql
type Attribute implements Node, ObjectWithMetadata {
  id: ID!
  productTypes(
    before: String
    after: String
    first: Int = 0
    last: Int = 0
  ): ProductTypeCountableConnection!
  productVariantTypes(
    before: String
    after: String
    first: Int = 0
    last: Int = 0
  ): ProductTypeCountableConnection!
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  inputType: AttributeInputTypeEnum
  name: String
  slug: String
  type: AttributeTypeEnum
  values: [AttributeValue]
  valueRequired: Boolean!
  visibleInStorefront: Boolean!
  filterableInStorefront: Boolean!
  filterableInDashboard: Boolean!
  availableInGrid: Boolean!
  translation(languageCode: LanguageCodeEnum!): AttributeTranslation
  storefrontSearchPosition: Int!
}
```

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](/api-reference/objects/metadata-item.mdx))

List of public metadata items. Can be accessed without permissions.

#### `inputType` ([`AttributeInputTypeEnum`](/api-reference/enums/attribute-input-type-enum.mdx))

The input type to use for entering attribute values in the dashboard.

#### `name` ([`String`](/api-reference/scalars/string.mdx))

Name of an attribute displayed in the interface.

#### `slug` ([`String`](/api-reference/scalars/string.mdx))

Internal representation of an attribute name.

#### `type` ([`AttributeTypeEnum`](/api-reference/enums/attribute-type-enum.mdx))

The attribute type.

#### `values` ([`[AttributeValue]`](/api-reference/objects/attribute-value.mdx))

List of attribute's values.

#### `valueRequired` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Whether the attribute requires values to be passed or not.

#### `visibleInStorefront` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Whether the attribute should be visible or not in storefront.

#### `filterableInStorefront` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Whether the attribute can be filtered in storefront.

#### `filterableInDashboard` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Whether the attribute can be filtered in dashboard.

#### `availableInGrid` ([`Boolean!`](/api-reference/scalars/boolean.mdx))

Whether the attribute can be displayed in the admin product list.

#### `translation` ([`AttributeTranslation`](/api-reference/objects/attribute-translation.mdx))

Returns translated attribute fields for the given language code.

- ##### `languageCode` ([`LanguageCodeEnum!`](/api-reference/enums/language-code-enum.mdx))

A language code to return the translation for attribute.

#### `storefrontSearchPosition` ([`Int!`](/api-reference/scalars/int.mdx))

The position of the attribute in the storefront navigation (0 by default).
