Represents a recipient of email notifications send by Saleor, such as notifications about new orders. Notifications can be assigned to staff users or arbitrary email addresses.

```graphql
type StaffNotificationRecipient implements Node {
  user: User
  active: Boolean
  id: ID!
  email: String
}
```

### Fields

#### `user` ([`User`](/api-reference/objects/user.mdx))

Returns a user subscribed to email notifications.

#### `active` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Determines if a notification active.

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `email` ([`String`](/api-reference/scalars/string.mdx))

Returns email address of a user subscribed to email notifications.
