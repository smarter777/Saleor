### Fields

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

The ID of the object.

#### `date` ([`DateTime`](/api-reference/scalars/date-time.mdx))

Date when event happened at in ISO 8601 format.

#### `type` ([`OrderEventsEnum`](/api-reference/enums/order-events-enum.mdx))

Order event type.

#### `user` ([`User`](/api-reference/objects/user.mdx))

User who performed the action.

#### `message` ([`String`](/api-reference/scalars/string.mdx))

Content of the event.

#### `email` ([`String`](/api-reference/scalars/string.mdx))

Email of the customer.

#### `emailType` ([`OrderEventsEmailsEnum`](/api-reference/enums/order-events-emails-enum.mdx))

Type of an email sent to the customer.

#### `amount` ([`Float`](/api-reference/scalars/float.mdx))

Amount of money.

#### `paymentId` ([`String`](/api-reference/scalars/string.mdx))

The payment ID from the payment gateway.

#### `paymentGateway` ([`String`](/api-reference/scalars/string.mdx))

The payment gateway of the payment.

#### `quantity` ([`Int`](/api-reference/scalars/int.mdx))

Number of items.

#### `composedId` ([`String`](/api-reference/scalars/string.mdx))

Composed ID of the Fulfillment.

#### `orderNumber` ([`String`](/api-reference/scalars/string.mdx))

User-friendly number of an order.

#### `invoiceNumber` ([`String`](/api-reference/scalars/string.mdx))

Number of an invoice related to the order.

#### `oversoldItems` ([`[String]`](/api-reference/scalars/string.mdx))

List of oversold lines names.

#### `lines` ([`[OrderEventOrderLineObject]`](/api-reference/objects/order-event-order-line-object.mdx))

The concerned lines.

#### `fulfilledItems` ([`[FulfillmentLine]`](/api-reference/objects/fulfillment-line.mdx))

The lines fulfilled.

#### `warehouse` ([`Warehouse`](/api-reference/objects/warehouse.mdx))

The warehouse were items were restocked.

#### `transactionReference` ([`String`](/api-reference/scalars/string.mdx))

The transaction reference of captured payment.

#### `shippingCostsIncluded` ([`Boolean`](/api-reference/scalars/boolean.mdx))

Define if shipping costs were included to the refund.
