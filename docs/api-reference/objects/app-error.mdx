No description

```graphql
type AppError {
  field: String
  message: String
  code: AppErrorCode!
  permissions: [PermissionEnum!]
}
```

### Fields

#### `field` ([`String`](/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`AppErrorCode!`](/api-reference/enums/app-error-code.mdx))

The error code.

#### `permissions` ([`[PermissionEnum!]`](/api-reference/enums/permission-enum.mdx))

List of permissions which causes the error.
