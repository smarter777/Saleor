\_Entity union as defined by Federation spec.

```graphql
union _Entity =
    App
  | Address
  | User
  | Group
  | ProductVariant
  | Product
  | ProductType
  | ProductMedia
  | Category
  | Collection
  | PageType
  | Order
```

### Possible types

#### [`App`](../../../api-reference/apps/objects/app) <Badge class="secondary" text="object"/> <Badge class="secondary" text="apps"/>

> Represents app data.

#### [`Address`](../../../api-reference/users/objects/address) <Badge class="secondary" text="object"/> <Badge class="secondary" text="users"/>

> Represents user address data.

#### [`User`](../../../api-reference/users/objects/user) <Badge class="secondary" text="object"/> <Badge class="secondary" text="users"/>

> Represents user data.

#### [`Group`](../../../api-reference/users/objects/group) <Badge class="secondary" text="object"/> <Badge class="secondary" text="users"/>

> Represents permission group data.

#### [`ProductVariant`](../../../api-reference/products/objects/product-variant) <Badge class="secondary" text="object"/> <Badge class="secondary" text="products"/>

> Represents a version of a product such as different size or color.

#### [`Product`](../../../api-reference/products/objects/product) <Badge class="secondary" text="object"/> <Badge class="secondary" text="products"/>

> Represents an individual item for sale in the storefront.

#### [`ProductType`](../../../api-reference/products/objects/product-type) <Badge class="secondary" text="object"/> <Badge class="secondary" text="products"/>

> Represents a type of product. It defines what attributes are available to products of this type.

#### [`ProductMedia`](../../../api-reference/products/objects/product-media) <Badge class="secondary" text="object"/> <Badge class="secondary" text="products"/>

> Represents a product media.

#### [`Category`](../../../api-reference/products/objects/category) <Badge class="secondary" text="object"/> <Badge class="secondary" text="products"/>

> Represents a single category of products. Categories allow to organize products in a tree-hierarchies which can be used for navigation in the storefront.

#### [`Collection`](../../../api-reference/products/objects/collection) <Badge class="secondary" text="object"/> <Badge class="secondary" text="products"/>

> Represents a collection of products.

#### [`PageType`](../../../api-reference/pages/objects/page-type) <Badge class="secondary" text="object"/> <Badge class="secondary" text="pages"/>

> Represents a type of page. It defines what attributes are available to pages of this type.

#### [`Order`](../../../api-reference/orders/objects/order) <Badge class="secondary" text="object"/> <Badge class="secondary" text="orders"/>

> Represents an order in the shop.
