Look up a menu item by ID.

```graphql
menuItem(
  id: ID!
  channel: String
): MenuItem
```

#### [`MenuItem`](../../../api-reference/miscellaneous/objects/menu-item) <Badge class="secondary" text="object"/> <Badge class="secondary" text="miscellaneous"/>
