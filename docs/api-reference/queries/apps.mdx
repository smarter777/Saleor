List of the apps.

```graphql
apps(
  filter: AppFilterInput
  sortBy: AppSortingInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): AppCountableConnection

```

### Arguments

#### `filter` ([`AppFilterInput`](/api-reference/inputs/app-filter-input.mdx))

Filtering options for apps.

#### `sortBy` ([`AppSortingInput`](/api-reference/inputs/app-sorting-input.mdx))

Sort apps.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`AppCountableConnection`](/api-reference/objects/app-countable-connection.mdx)
