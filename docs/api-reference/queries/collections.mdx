List of the shop's collections.

```graphql
collections(
  filter: CollectionFilterInput
  sortBy: CollectionSortingInput
  channel: String
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): CollectionCountableConnection

```

### Arguments

#### `filter` ([`CollectionFilterInput`](/api-reference/inputs/collection-filter-input.mdx))

Filtering options for collections.

#### `sortBy` ([`CollectionSortingInput`](/api-reference/inputs/collection-sorting-input.mdx))

Sort collections.

#### `channel` ([`String`](/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`CollectionCountableConnection`](/api-reference/objects/collection-countable-connection.mdx)
