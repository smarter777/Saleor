List of the shop's products.

```graphql
products(
  filter: ProductFilterInput
  sortBy: ProductOrder
  stockAvailability: StockAvailability
  channel: String
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): ProductCountableConnection

```

### Arguments

#### `filter` ([`ProductFilterInput`](/api-reference/inputs/product-filter-input.mdx))

Filtering options for products.

#### `sortBy` ([`ProductOrder`](/api-reference/inputs/product-order.mdx))

Sort products.

#### `stockAvailability` ([`StockAvailability`](/api-reference/enums/stock-availability.mdx))

[Deprecated] Filter products by stock availability. Use the `filter` field instead. This field will be removed after 2020-07-31.

#### `channel` ([`String`](/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`ProductCountableConnection`](/api-reference/objects/product-countable-connection.mdx)
