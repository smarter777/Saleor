Look up a export file by ID.

```graphql
exportFile(
  id: ID!
): ExportFile

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the export file job.

### Type

#### [`ExportFile`](/api-reference/objects/export-file.mdx)

Represents a job data of exported file.
