Look up a navigation menu by ID or name.

```graphql
menu(
  channel: String
  id: ID = 0
  name: String
  slug: String
): Menu

```

### Arguments

#### `channel` ([`String`](/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `id` ([`ID`](/api-reference/scalars/id.mdx))

ID of the menu.

#### `name` ([`String`](/api-reference/scalars/string.mdx))

The menu's name.

#### `slug` ([`String`](/api-reference/scalars/string.mdx))

The menu's slug.

### Type

#### [`Menu`](/api-reference/objects/menu.mdx)

Represents a single menu - an object that is used to help navigate through the store.
