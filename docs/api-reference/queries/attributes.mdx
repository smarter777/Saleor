List of the shop's attributes.

```graphql
attributes(
  filter: AttributeFilterInput
  sortBy: AttributeSortingInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): AttributeCountableConnection

```

### Arguments

#### `filter` ([`AttributeFilterInput`](/api-reference/inputs/attribute-filter-input.mdx))

Filtering options for attributes.

#### `sortBy` ([`AttributeSortingInput`](/api-reference/inputs/attribute-sorting-input.mdx))

Sorting options for attributes.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`AttributeCountableConnection`](/api-reference/objects/attribute-countable-connection.mdx)
