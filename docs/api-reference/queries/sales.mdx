List of the shop's sales.

```graphql
sales(
  filter: SaleFilterInput
  sortBy: SaleSortingInput
  query: String
  channel: String
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): SaleCountableConnection

```

### Arguments

#### `filter` ([`SaleFilterInput`](/api-reference/inputs/sale-filter-input.mdx))

Filtering options for sales.

#### `sortBy` ([`SaleSortingInput`](/api-reference/inputs/sale-sorting-input.mdx))

Sort sales.

#### `query` ([`String`](/api-reference/scalars/string.mdx))

Search sales by name, value or type.

#### `channel` ([`String`](/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.
