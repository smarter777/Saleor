Look up a checkout by token and slug of channel.

```graphql
checkout(
  token: UUID
): Checkout

```

### Arguments

#### `token` ([`UUID`](/api-reference/scalars/uuid.mdx))

The checkout's token.

### Type

#### [`Checkout`](/api-reference/objects/checkout.mdx)

Checkout object.
