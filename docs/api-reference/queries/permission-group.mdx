Look up permission group by ID.

```graphql
permissionGroup(
  id: ID!
): Group

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the group.

### Type

#### [`Group`](/api-reference/objects/group.mdx)

Represents permission group data.
