List of the shop's product types.

```graphql
productTypes(
  filter: ProductTypeFilterInput
  sortBy: ProductTypeSortingInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): ProductTypeCountableConnection

```

### Arguments

#### `filter` ([`ProductTypeFilterInput`](/api-reference/inputs/product-type-filter-input.mdx))

Filtering options for product types.

#### `sortBy` ([`ProductTypeSortingInput`](/api-reference/inputs/product-type-sorting-input.mdx))

Sort product types.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.
