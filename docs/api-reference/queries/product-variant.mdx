Look up a product variant by ID or SKU.

```graphql
productVariant(
  id: ID = 0
  sku: String
  channel: String
): ProductVariant

```

### Arguments

#### `id` ([`ID`](/api-reference/scalars/id.mdx))

ID of the product variant.

#### `sku` ([`String`](/api-reference/scalars/string.mdx))

Sku of the product variant.

#### `channel` ([`String`](/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

### Type

#### [`ProductVariant`](/api-reference/objects/product-variant.mdx)

Represents a version of a product such as different size or color.
