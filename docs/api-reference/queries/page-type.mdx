Look up a page type by ID.

```graphql
pageType(
  id: ID!
): PageType

```

### Arguments

#### `id` ([`ID!`](/api-reference/scalars/id.mdx))

ID of the page type.

### Type

#### [`PageType`](/api-reference/objects/page-type.mdx)

Represents a type of page. It defines what attributes are available to pages of this type.
