List of the shop's categories.

```graphql
categories(
  filter: CategoryFilterInput
  sortBy: CategorySortingInput
  level: Int = 0
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): CategoryCountableConnection

```

### Arguments

#### `filter` ([`CategoryFilterInput`](/api-reference/inputs/category-filter-input.mdx))

Filtering options for categories.

#### `sortBy` ([`CategorySortingInput`](/api-reference/inputs/category-sorting-input.mdx))

Sort categories.

#### `level` ([`Int`](/api-reference/scalars/int.mdx))

Filter categories by the nesting level in the category tree.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`CategoryCountableConnection`](/api-reference/objects/category-countable-connection.mdx)
