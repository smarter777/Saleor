List of the page types.

```graphql
pageTypes(
  sortBy: PageTypeSortingInput
  filter: PageTypeFilterInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): PageTypeCountableConnection

```

### Arguments

#### `sortBy` ([`PageTypeSortingInput`](/api-reference/inputs/page-type-sorting-input.mdx))

Sort page types.

#### `filter` ([`PageTypeFilterInput`](/api-reference/inputs/page-type-filter-input.mdx))

Filtering options for page types.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`PageTypeCountableConnection`](/api-reference/objects/page-type-countable-connection.mdx)
