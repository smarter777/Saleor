Look up an order by token.

```graphql
orderByToken(
  token: UUID!
): Order

```

### Arguments

#### `token` ([`UUID!`](/api-reference/scalars/uuid.mdx))

The order's token.

### Type

#### [`Order`](/api-reference/objects/order.mdx)

Represents an order in the shop.
