List of permission groups.

```graphql
permissionGroups(
  filter: PermissionGroupFilterInput
  sortBy: PermissionGroupSortingInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): GroupCountableConnection

```

### Arguments

#### `filter` ([`PermissionGroupFilterInput`](/api-reference/inputs/permission-group-filter-input.mdx))

Filtering options for permission groups.

#### `sortBy` ([`PermissionGroupSortingInput`](/api-reference/inputs/permission-group-sorting-input.mdx))

Sort permission groups.

#### `before` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`GroupCountableConnection`](/api-reference/objects/group-countable-connection.mdx)
