Represents a delivery method chosen for the checkout. `Warehouse` type is used when checkout is marked as "click and collect" and `ShippingMethod` otherwise.

Added in Saleor 3.1.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

```graphql
union DeliveryMethod = Warehouse | ShippingMethod
```


### Possible types

#### [`Warehouse`](../../../developer/api-reference/objects/warehouse) <Badge class="secondary" text="object"/>
> Represents warehouse.
> 

#### [`ShippingMethod`](../../../developer/api-reference/objects/shipping-method) <Badge class="secondary" text="object"/>
> Shipping methods that can be used as means of shipping for orders and checkouts.
>
