Look up a category by ID or slug.

```graphql
category(
  id: ID = 0
  slug: String
): Category

```

### Arguments

#### `id` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the category.

#### `slug` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of the category

### Type

#### [`Category`](/developer/api-reference/objects/category.mdx)

Represents a single category of products. Categories allow to organize products in a tree-hierarchies which can be used for navigation in the storefront.
