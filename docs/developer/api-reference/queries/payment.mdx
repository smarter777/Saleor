Look up a payment by ID.

```graphql
payment(
  id: ID!
): Payment

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the payment.

### Type

#### [`Payment`](/developer/api-reference/objects/payment.mdx)

Represents a payment of a given type.
