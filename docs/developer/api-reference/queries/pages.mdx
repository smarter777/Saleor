List of the shop's pages.

```graphql
pages(
  sortBy: PageSortingInput
  filter: PageFilterInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): PageCountableConnection

```

### Arguments

#### `sortBy` ([`PageSortingInput`](/developer/api-reference/inputs/page-sorting-input.mdx))

Sort pages.

#### `filter` ([`PageFilterInput`](/developer/api-reference/inputs/page-filter-input.mdx))

Filtering options for pages.

#### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`PageCountableConnection`](/developer/api-reference/objects/page-countable-connection.mdx)
