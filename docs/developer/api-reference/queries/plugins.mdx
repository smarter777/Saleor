List of plugins.

```graphql
plugins(
  filter: PluginFilterInput
  sortBy: PluginSortingInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): PluginCountableConnection

```

### Arguments

#### `filter` ([`PluginFilterInput`](/developer/api-reference/inputs/plugin-filter-input.mdx))

Filtering options for plugins.

#### `sortBy` ([`PluginSortingInput`](/developer/api-reference/inputs/plugin-sorting-input.mdx))

Sort plugins.

#### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`PluginCountableConnection`](/developer/api-reference/objects/plugin-countable-connection.mdx)
