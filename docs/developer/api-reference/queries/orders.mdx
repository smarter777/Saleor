List of orders.

```graphql
orders(
  sortBy: OrderSortingInput
  filter: OrderFilterInput
  created: ReportingPeriod
  status: OrderStatusFilter
  channel: String
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): OrderCountableConnection

```

### Arguments

#### `sortBy` ([`OrderSortingInput`](/developer/api-reference/inputs/order-sorting-input.mdx))

Sort orders.

#### `filter` ([`OrderFilterInput`](/developer/api-reference/inputs/order-filter-input.mdx))

Filtering options for orders.

#### `created` ([`ReportingPeriod`](/developer/api-reference/enums/reporting-period.mdx))

[Deprecated] Filter orders from a selected timespan. Use the `filter` field instead. This field will be removed after 2020-07-31.

#### `status` ([`OrderStatusFilter`](/developer/api-reference/enums/order-status-filter.mdx))

[Deprecated] Filter order by status. Use the `filter` field instead. This field will be removed after 2020-07-31.

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`OrderCountableConnection`](/developer/api-reference/objects/order-countable-connection.mdx)
