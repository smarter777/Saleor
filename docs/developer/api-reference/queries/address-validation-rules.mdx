Returns address validation rules.

```graphql
addressValidationRules(
  countryCode: CountryCode!
  countryArea: String
  city: String
  cityArea: String
): AddressValidationData

```

### Arguments

#### `countryCode` ([`CountryCode!`](/developer/api-reference/enums/country-code.mdx))

Two-letter ISO 3166-1 country code.

#### `countryArea` ([`String`](/developer/api-reference/scalars/string.mdx))

Designation of a region, province or state.

#### `city` ([`String`](/developer/api-reference/scalars/string.mdx))

City or a town name.

#### `cityArea` ([`String`](/developer/api-reference/scalars/string.mdx))

Sublocality like a district.

### Type

#### [`AddressValidationData`](/developer/api-reference/objects/address-validation-data.mdx)
