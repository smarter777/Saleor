List of the shop's product types.

```graphql
productTypes(
  filter: ProductTypeFilterInput
  sortBy: ProductTypeSortingInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): ProductTypeCountableConnection

```

### Arguments

#### `filter` ([`ProductTypeFilterInput`](/developer/api-reference/inputs/product-type-filter-input.mdx))

Filtering options for product types.

#### `sortBy` ([`ProductTypeSortingInput`](/developer/api-reference/inputs/product-type-sorting-input.mdx))

Sort product types.

#### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`ProductTypeCountableConnection`](/developer/api-reference/objects/product-type-countable-connection.mdx)
