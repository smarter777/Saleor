Look up a checkout line by ID.

```graphql
checkoutLine(
  id: ID = 0
): CheckoutLine

```

### Arguments

#### `id` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the checkout line.

### Type

#### [`CheckoutLine`](/developer/api-reference/objects/checkout-line.mdx)

Represents an item in the checkout.
