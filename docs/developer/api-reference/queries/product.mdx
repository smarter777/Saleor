Look up a product by ID.

```graphql
product(
  id: ID = 0
  slug: String
  channel: String
): Product

```

### Arguments

#### `id` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the product.

#### `slug` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of the product.

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

### Type

#### [`Product`](/developer/api-reference/objects/product.mdx)

Represents an individual item for sale in the storefront.
