Look up an address by ID.

```graphql
address(
  id: ID!
): Address

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an address.

### Type

#### [`Address`](/developer/api-reference/objects/address.mdx)

Represents user address data.
