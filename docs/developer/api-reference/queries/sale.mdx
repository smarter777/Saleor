Look up a sale by ID.

```graphql
sale(
  id: ID!
  channel: String
): Sale

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the sale.

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

### Type

#### [`Sale`](/developer/api-reference/objects/sale.mdx)

Sales allow creating discounts for categories, collections or products and are visible to all the customers.
