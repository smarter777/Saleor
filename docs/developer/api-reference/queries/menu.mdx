Look up a navigation menu by ID or name.

```graphql
menu(
  channel: String
  id: ID = 0
  name: String
  slug: String
): Menu

```

### Arguments

#### `channel` ([`String`](/developer/api-reference/scalars/string.mdx))

Slug of a channel for which the data should be returned.

#### `id` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the menu.

#### `name` ([`String`](/developer/api-reference/scalars/string.mdx))

The menu's name.

#### `slug` ([`String`](/developer/api-reference/scalars/string.mdx))

The menu's slug.

### Type

#### [`Menu`](/developer/api-reference/objects/menu.mdx)

Represents a single menu - an object that is used to help navigate through the store.
