Look up a channel by ID.

```graphql
channel(
  id: ID = 0
): Channel

```

### Arguments

#### `id` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the channel.

### Type

#### [`Channel`](/developer/api-reference/objects/channel.mdx)

Represents channel.
