Look up a warehouse by ID.

Requires one of the following permissions: MANAGE_PRODUCTS, MANAGE_ORDERS, MANAGE_SHIPPING.

```graphql
warehouse(
  id: ID!
): Warehouse
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an warehouse

### Type

#### [`Warehouse`](../../../developer/api-reference/objects/warehouse)

Represents warehouse.
