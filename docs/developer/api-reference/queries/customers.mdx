List of the shop's customers.

```graphql
customers(
  filter: CustomerFilterInput
  sortBy: UserSortingInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): UserCountableConnection

```

### Arguments

#### `filter` ([`CustomerFilterInput`](/developer/api-reference/inputs/customer-filter-input.mdx))

Filtering options for customers.

#### `sortBy` ([`UserSortingInput`](/developer/api-reference/inputs/user-sorting-input.mdx))

Sort customers.

#### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`UserCountableConnection`](/developer/api-reference/objects/user-countable-connection.mdx)
