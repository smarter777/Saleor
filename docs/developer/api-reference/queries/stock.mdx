Look up a stock by ID

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
stock(
  id: ID!
): Stock
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an warehouse

### Type

#### [`Stock`](../../../developer/api-reference/objects/stock)

Represents stock.
