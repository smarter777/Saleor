Look up a user by ID or email address.

```graphql
user(
  id: ID = 0
  email: String
): User

```

### Arguments

#### `id` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the user.

#### `email` ([`String`](/developer/api-reference/scalars/string.mdx))

Email address of the user.

### Type

#### [`User`](/developer/api-reference/objects/user.mdx)

Represents user data.
