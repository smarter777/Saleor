Look up a page by ID or slug.

```graphql
page(
  id: ID
  slug: String
): Page
```

#### [<code style={{ fontWeight: 'normal' }}>page.<b>id</b></code>](#)<Bullet />[`ID`](../../../developer/api-reference/scalars/id) <Badge class="secondary" text="scalar"/>

> 

#### [<code style={{ fontWeight: 'normal' }}>page.<b>slug</b></code>](#)<Bullet />[`String`](../../../developer/api-reference/scalars/string) <Badge class="secondary" text="scalar"/>

> 

### Type
