List of export files.

```graphql
exportFiles(
  filter: ExportFileFilterInput
  sortBy: ExportFileSortingInput
  before: String
  after: String
  first: Int = 0
  last: Int = 0
): ExportFileCountableConnection

```

### Arguments

#### `filter` ([`ExportFileFilterInput`](/developer/api-reference/inputs/export-file-filter-input.mdx))

Filtering options for export files.

#### `sortBy` ([`ExportFileSortingInput`](/developer/api-reference/inputs/export-file-sorting-input.mdx))

Sort export files.

#### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

#### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

#### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

### Type

#### [`ExportFileCountableConnection`](/developer/api-reference/objects/export-file-countable-connection.mdx)
