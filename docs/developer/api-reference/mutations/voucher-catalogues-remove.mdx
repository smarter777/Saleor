Removes products, categories, collections from a voucher.

```graphql
voucherCataloguesRemove(
  id: ID!
  input: CatalogueInput!
): VoucherRemoveCatalogues

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a voucher.

#### `input` ([`CatalogueInput!`](/developer/api-reference/inputs/catalogue-input.mdx))

Fields required to modify catalogue IDs of voucher.

### Type

#### [`VoucherRemoveCatalogues`](/developer/api-reference/objects/voucher-remove-catalogues.mdx)

Removes products, categories, collections from a voucher.
