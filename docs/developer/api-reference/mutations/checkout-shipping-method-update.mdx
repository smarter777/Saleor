Updates the shipping address of the checkout.

```graphql
checkoutShippingMethodUpdate(
  checkoutId: ID = 0
  shippingMethodId: ID!
): CheckoutShippingMethodUpdate

```

### Arguments

#### `checkoutId` ([`ID`](/developer/api-reference/scalars/id.mdx))

Checkout ID.

#### `shippingMethodId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

Shipping method.

### Type

#### [`CheckoutShippingMethodUpdate`](/developer/api-reference/objects/checkout-shipping-method-update.mdx)

Updates the shipping address of the checkout.
