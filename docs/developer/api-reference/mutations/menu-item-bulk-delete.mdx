Deletes menu items.

Requires one of the following permissions: MANAGE_MENUS.

```graphql
menuItemBulkDelete(
  ids: [ID!]!
): MenuItemBulkDelete
```

### Arguments

#### [`ids`](#) ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of menu item IDs to delete.

### Type

#### [`MenuItemBulkDelete`](../../../developer/api-reference/objects/menu-item-bulk-delete)

Deletes menu items.

Requires one of the following permissions: MANAGE_MENUS.
