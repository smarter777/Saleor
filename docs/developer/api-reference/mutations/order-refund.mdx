Refund an order.

```graphql
orderRefund(
  amount: PositiveDecimal!
  id: ID!
): OrderRefund

```

### Arguments

#### `amount` ([`PositiveDecimal!`](/developer/api-reference/scalars/positive-decimal.mdx))

Amount of money to refund.

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the order to refund.

### Type

#### [`OrderRefund`](/developer/api-reference/objects/order-refund.mdx)

Refund an order.
