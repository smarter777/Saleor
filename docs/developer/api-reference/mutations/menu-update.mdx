Updates a menu.

```graphql
menuUpdate(
  id: ID!
  input: MenuInput!
): MenuUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a menu to update.

#### `input` ([`MenuInput!`](/developer/api-reference/inputs/menu-input.mdx))

Fields required to update a menu.

### Type

#### [`MenuUpdate`](/developer/api-reference/objects/menu-update.mdx)

Updates a menu.
