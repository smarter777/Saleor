---

Remove shipping zone from given warehouse.

```graphql
unassignWarehouseShippingZone(
  id: ID!
  shippingZoneIds: [ID!]!
): WarehouseShippingZoneUnassign

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a warehouse to update.

#### `shippingZoneIds` ([`[ID!]!`](/developer/api-reference/scalars/id.mdx))

List of shipping zone IDs.

### Type

#### [`WarehouseShippingZoneUnassign`](/developer/api-reference/objects/warehouse-shipping-zone-unassign.mdx)

Remove shipping zone from given warehouse.
