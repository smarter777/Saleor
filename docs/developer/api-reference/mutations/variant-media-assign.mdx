Assign an media to a product variant.

```graphql
variantMediaAssign(
  mediaId: ID!
  variantId: ID!
): VariantMediaAssign

```

### Arguments

#### `mediaId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product media to assign to a variant.

#### `variantId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product variant.

### Type

#### [`VariantMediaAssign`](/developer/api-reference/objects/variant-media-assign.mdx)

Assign an media to a product variant.
