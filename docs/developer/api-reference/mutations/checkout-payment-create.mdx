Create a new payment for given checkout.

```graphql
checkoutPaymentCreate(
  checkoutId: ID!
  input: PaymentInput!
): CheckoutPaymentCreate

```

### Arguments

#### `checkoutId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

Checkout ID.

#### `input` ([`PaymentInput!`](/developer/api-reference/inputs/payment-input.mdx))

Data required to create a new payment.

### Type

#### [`CheckoutPaymentCreate`](/developer/api-reference/objects/checkout-payment-create.mdx)

Create a new payment for given checkout.
