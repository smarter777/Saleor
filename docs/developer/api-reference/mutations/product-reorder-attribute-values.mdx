Reorder product attribute values.

```graphql
productReorderAttributeValues(
  attributeId: ID!
  moves: [ReorderInput]!
  productId: ID!
): ProductReorderAttributeValues

```

### Arguments

#### `attributeId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an attribute.

#### `moves` ([`[ReorderInput]!`](/developer/api-reference/inputs/reorder-input.mdx))

The list of reordering operations for given attribute values.

#### `productId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product.

### Type

#### [`ProductReorderAttributeValues`](/developer/api-reference/objects/product-reorder-attribute-values.mdx)

Reorder product attribute values.
