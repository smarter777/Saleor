Remove user account.

Requires one of the following permissions: AUTHENTICATED_USER.

```graphql
accountDelete(
  token: String!
): AccountDelete
```

### Arguments

#### [`token`](#) ([`String!`](../../../developer/api-reference/scalars/string))

A one-time token required to remove account. Sent by email using AccountRequestDeletion mutation.

### Type

#### [`AccountDelete`](../../../developer/api-reference/objects/account-delete)

Remove user account.

Requires one of the following permissions: AUTHENTICATED_USER.
