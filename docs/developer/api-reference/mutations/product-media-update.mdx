Updates a product media.

```graphql
productMediaUpdate(
  id: ID!
  input: ProductMediaUpdateInput!
): ProductMediaUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product media to update.

#### `input` ([`ProductMediaUpdateInput!`](/developer/api-reference/inputs/product-media-update-input.mdx))

Fields required to update a product media.

### Type

#### [`ProductMediaUpdate`](/developer/api-reference/objects/product-media-update.mdx)

Updates a product media.
