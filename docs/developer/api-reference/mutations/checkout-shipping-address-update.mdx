Update shipping address in the existing checkout.

```graphql
checkoutShippingAddressUpdate(
  checkoutId: ID!
  shippingAddress: AddressInput!
): CheckoutShippingAddressUpdate

```

### Arguments

#### `checkoutId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the checkout.

#### `shippingAddress` ([`AddressInput!`](/developer/api-reference/inputs/address-input.mdx))

The mailing address to where the checkout will be shipped.

### Type

#### [`CheckoutShippingAddressUpdate`](/developer/api-reference/objects/checkout-shipping-address-update.mdx)

Update shipping address in the existing checkout.
