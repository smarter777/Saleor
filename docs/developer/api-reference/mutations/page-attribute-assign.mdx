Assign attributes to a given page type.

```graphql
pageAttributeAssign(
  attributeIds: [ID!]!
  pageTypeId: ID!
): PageAttributeAssign

```

### Arguments

#### `attributeIds` ([`[ID!]!`](/developer/api-reference/scalars/id.mdx))

The IDs of the attributes to assign.

#### `pageTypeId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the page type to assign the attributes into.

### Type

#### [`PageAttributeAssign`](/developer/api-reference/objects/page-attribute-assign.mdx)

Assign attributes to a given page type.
