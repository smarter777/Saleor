Obtain external access tokens for user by custom plugin.

```graphql
externalObtainAccessTokens(
  input: JSONString!
  pluginId: String!
): ExternalObtainAccessTokens
```

### Arguments

#### [`input`](#) ([`JSONString!`](../../../developer/api-reference/scalars/jsonstring))

The data required by plugin to create authentication data.

#### [`pluginId`](#) ([`String!`](../../../developer/api-reference/scalars/string))

The ID of the authentication plugin.

### Type

#### [`ExternalObtainAccessTokens`](../../../developer/api-reference/objects/external-obtain-access-tokens)

Obtain external access tokens for user by custom plugin.
