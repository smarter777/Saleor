Changes ordering of the product media.

```graphql
productMediaReorder(
  mediaIds: [ID]!
  productId: ID!
): ProductMediaReorder

```

### Arguments

#### `mediaIds` ([`[ID]!`](/developer/api-reference/scalars/id.mdx))

IDs of a product media in the desired order.

#### `productId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of product that media order will be altered.

### Type

#### [`ProductMediaReorder`](/developer/api-reference/objects/product-media-reorder.mdx)

Changes ordering of the product media.
