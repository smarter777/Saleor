Updates an order line of a draft order.

```graphql
draftOrderLineUpdate(
  id: ID!
  input: OrderLineInput!
): DraftOrderLineUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the order line to update.

#### `input` ([`OrderLineInput!`](/developer/api-reference/inputs/order-line-input.mdx))

Fields required to update an order line.

### Type

#### [`DraftOrderLineUpdate`](/developer/api-reference/objects/draft-order-line-update.mdx)

Updates an order line of a draft order.
