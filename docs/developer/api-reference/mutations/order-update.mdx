Updates an order.

```graphql
orderUpdate(
  id: ID!
  input: OrderUpdateInput!
): OrderUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an order to update.

#### `input` ([`OrderUpdateInput!`](/developer/api-reference/inputs/order-update-input.mdx))

Fields required to update an order.

### Type

#### [`OrderUpdate`](/developer/api-reference/objects/order-update.mdx)

Updates an order.
