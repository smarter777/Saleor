Creates user address.

```graphql
addressCreate(
  input: AddressInput!
  userId: ID!
): AddressCreate

```

### Arguments

#### `input` ([`AddressInput!`](/developer/api-reference/inputs/address-input.mdx))

Fields required to create address.

#### `userId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a user to create address for.

### Type

#### [`AddressCreate`](/developer/api-reference/objects/address-create.mdx)

Creates user address.
