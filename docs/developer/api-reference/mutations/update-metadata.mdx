Updates metadata of an object.

```graphql
updateMetadata(
  id: ID!
  input: [MetadataInput!]!
): UpdateMetadata

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an object to update.

#### `input` ([`[MetadataInput!]!`](/developer/api-reference/inputs/metadata-input.mdx))

Fields required to update the object's metadata.

### Type

#### [`UpdateMetadata`](/developer/api-reference/objects/update-metadata.mdx)

Updates metadata of an object.
