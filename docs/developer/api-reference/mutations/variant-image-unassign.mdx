Unassign an image from a product variant.

```graphql
variantImageUnassign(
  imageId: ID!
  variantId: ID!
): VariantImageUnassign

```

### Arguments

#### `imageId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product image to unassign from a variant.

#### `variantId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product variant.

### Type

#### [`VariantImageUnassign`](/developer/api-reference/objects/variant-image-unassign.mdx)

Unassign an image from a product variant.
