Adds products, categories, collections to a voucher.

```graphql
voucherCataloguesAdd(
  id: ID!
  input: CatalogueInput!
): VoucherAddCatalogues

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a voucher.

#### `input` ([`CatalogueInput!`](/developer/api-reference/inputs/catalogue-input.mdx))

Fields required to modify catalogue IDs of voucher.

### Type

#### [`VoucherAddCatalogues`](/developer/api-reference/objects/voucher-add-catalogues.mdx)

Adds products, categories, collections to a voucher.
