Request an invoice for the order using plugin.

```graphql
invoiceRequest(
  number: String
  orderId: ID!
): InvoiceRequest

```

### Arguments

#### `number` ([`String`](/developer/api-reference/scalars/string.mdx))

Invoice number, if not provided it will be generated.

#### `orderId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the order related to invoice.

### Type

#### [`InvoiceRequest`](/developer/api-reference/objects/invoice-request.mdx)

Request an invoice for the order using plugin.
