Deletes a shipping price.

```graphql
shippingPriceDelete(

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a shipping price to delete.

### Type

#### [`ShippingPriceDelete`](/developer/api-reference/objects/shipping-price-delete.mdx)

Deletes a shipping price.
