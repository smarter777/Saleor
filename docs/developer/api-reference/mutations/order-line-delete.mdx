Deletes an order line from an order.

```graphql
orderLineDelete(
  id: ID!
): OrderLineDelete

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the order line to delete.

### Type

#### [`OrderLineDelete`](/developer/api-reference/objects/order-line-delete.mdx)

Deletes an order line from an order.
