Reorder the attributes of a page type.

Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.

```graphql
pageTypeReorderAttributes(
  moves: [ReorderInput!]!
  pageTypeId: ID!
): PageTypeReorderAttributes
```

### Arguments

#### [`moves`](#) ([`[ReorderInput!]!`](../../../developer/api-reference/inputs/reorder-input))

The list of attribute reordering operations.

#### [`pageTypeId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a page type.

### Type

#### [`PageTypeReorderAttributes`](../../../developer/api-reference/objects/page-type-reorder-attributes)

Reorder the attributes of a page type.

Requires one of the following permissions: MANAGE_PAGE_TYPES_AND_ATTRIBUTES.
