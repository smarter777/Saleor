Unassign an media from a product variant.

```graphql
variantMediaUnassign(
  mediaId: ID!
  variantId: ID!
): VariantMediaUnassign

```

### Arguments

#### `mediaId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product media to unassign from a variant.

#### `variantId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product variant.

### Type

#### [`VariantMediaUnassign`](/developer/api-reference/objects/variant-media-unassign.mdx)

Unassign an media from a product variant.
