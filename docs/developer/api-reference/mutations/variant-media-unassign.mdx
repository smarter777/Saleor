Unassign an media from a product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
variantMediaUnassign(
  mediaId: ID!
  variantId: ID!
): VariantMediaUnassign
```

### Arguments

#### [`mediaId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a product media to unassign from a variant.

#### [`variantId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a product variant.

### Type

#### [`VariantMediaUnassign`](../../../developer/api-reference/objects/variant-media-unassign)

Unassign an media from a product variant.

Requires one of the following permissions: MANAGE_PRODUCTS.
