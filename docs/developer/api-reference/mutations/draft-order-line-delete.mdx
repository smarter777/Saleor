Deletes an order line from a draft order.

```graphql
draftOrderLineDelete(
  id: ID!
): DraftOrderLineDelete

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the order line to delete.

### Type

#### [`DraftOrderLineDelete`](/developer/api-reference/objects/draft-order-line-delete.mdx)

Deletes an order line from a draft order.
