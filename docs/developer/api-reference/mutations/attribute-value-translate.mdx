Creates/Updates translations for attribute value.

```graphql
attributeValueTranslate(

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

Attribute Value ID.

#### `input` ([`AttributeValueTranslationInput!`](/developer/api-reference/inputs/attribute-value-translation-input.mdx))

#### `languageCode` ([`LanguageCodeEnum!`](/developer/api-reference/enums/language-code-enum.mdx))

Translation language code.

### Type

#### [`AttributeValueTranslate`](/developer/api-reference/objects/attribute-value-translate.mdx)

Creates/Updates translations for attribute value.
