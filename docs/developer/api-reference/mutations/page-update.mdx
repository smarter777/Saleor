Updates an existing page.

```graphql
pageUpdate(
  id: ID!
  input: PageInput!
): PageUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a page to update.

#### `input` ([`PageInput!`](/developer/api-reference/inputs/page-input.mdx))

Fields required to update a page.

### Type

#### [`PageUpdate`](/developer/api-reference/objects/page-update.mdx)

Updates an existing page.
