Adds discount to the order.

```graphql
orderDiscountAdd(
  input: OrderDiscountCommonInput!
  orderId: ID!
): OrderDiscountAdd

```

### Arguments

#### `input` ([`OrderDiscountCommonInput!`](/developer/api-reference/inputs/order-discount-common-input.mdx))

Fields required to create a discount for the order.

#### `orderId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an order to discount.

### Type

#### [`OrderDiscountAdd`](/developer/api-reference/objects/order-discount-add.mdx)

Adds discount to the order.
