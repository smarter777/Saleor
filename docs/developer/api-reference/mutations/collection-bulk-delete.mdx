Deletes collections.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
collectionBulkDelete(
  ids: [ID!]!
): CollectionBulkDelete
```

### Arguments

#### [`ids`](#) ([`[ID!]!`](../../../developer/api-reference/scalars/id))

List of collection IDs to delete.

### Type

#### [`CollectionBulkDelete`](../../../developer/api-reference/objects/collection-bulk-delete)

Deletes collections.

Requires one of the following permissions: MANAGE_PRODUCTS.
