Creates a new category.

```graphql
categoryCreate(
  input: CategoryInput!
  parent: ID = 0
): CategoryCreate

```

### Arguments

#### `input` ([`CategoryInput!`](/developer/api-reference/inputs/category-input.mdx))

Fields required to create a category.

#### `parent` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the parent category. If empty, category will be top level category.

### Type

#### [`CategoryCreate`](/developer/api-reference/objects/category-create.mdx)

Creates a new category.
