Creates a new category. 

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
categoryCreate(
  input: CategoryInput!
  parent: ID
): CategoryCreate
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>categoryCreate.<b>input</b></code>](#)<Bullet />[`CategoryInput!`](../../../developer/api-reference/inputs/category-input) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="input"/>
> Fields required to create a category.
> 

#### [<code style={{ fontWeight: 'normal' }}>categoryCreate.<b>parent</b></code>](#)<Bullet />[`ID`](../../../developer/api-reference/scalars/id) <Badge class="secondary" text="scalar"/>
> ID of the parent category. If empty, category will be top level category.
