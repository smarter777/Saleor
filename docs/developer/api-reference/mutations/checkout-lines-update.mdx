Updates checkout line in the existing checkout.

```graphql
checkoutLinesUpdate(
  checkoutId: ID!
  lines: [CheckoutLineInput]!
): CheckoutLinesUpdate

```

### Arguments

#### `checkoutId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the checkout.

#### `lines` ([`[CheckoutLineInput]!`](/developer/api-reference/inputs/checkout-line-input.mdx))

A list of checkout lines, each containing information about an item in the checkout.

### Type

#### [`CheckoutLinesUpdate`](/developer/api-reference/objects/checkout-lines-update.mdx)

Updates checkout line in the existing checkout.
