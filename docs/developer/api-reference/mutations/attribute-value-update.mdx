Updates value of an attribute.

```graphql
attributeValueUpdate(
  id: ID!
  input: AttributeValueCreateInput!
): AttributeValueUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an AttributeValue to update.

#### `input` ([`AttributeValueCreateInput!`](/developer/api-reference/inputs/attribute-value-create-input.mdx))

Fields required to update an AttributeValue.

### Type

#### [`AttributeValueUpdate`](/developer/api-reference/objects/attribute-value-update.mdx)

Updates value of an attribute.
