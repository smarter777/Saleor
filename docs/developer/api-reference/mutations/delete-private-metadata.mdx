Delete object's private metadata.

```graphql
deletePrivateMetadata(
  id: ID!
  keys: [String!]!
): DeletePrivateMetadata

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an object to update.

#### `keys` ([`[String!]!`](/developer/api-reference/scalars/string.mdx))

Metadata keys to delete.

### Type

#### [`DeletePrivateMetadata`](/developer/api-reference/objects/delete-private-metadata.mdx)

Delete object's private metadata.
