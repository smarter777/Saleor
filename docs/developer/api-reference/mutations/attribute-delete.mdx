Deletes an attribute.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.

```graphql
attributeDelete(
  id: ID!
): AttributeDelete
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of an attribute to delete.

### Type

#### [`AttributeDelete`](../../../developer/api-reference/objects/attribute-delete)

Deletes an attribute.

Requires one of the following permissions: MANAGE_PRODUCT_TYPES_AND_ATTRIBUTES.
