Update language code in the existing checkout.

```graphql
checkoutLanguageCodeUpdate(
  checkoutId: ID!
  languageCode: LanguageCodeEnum!
): CheckoutLanguageCodeUpdate

```

### Arguments

#### `checkoutId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the checkout.

#### `languageCode` ([`LanguageCodeEnum!`](/developer/api-reference/enums/language-code-enum.mdx))

New language code.

### Type

#### [`CheckoutLanguageCodeUpdate`](/developer/api-reference/objects/checkout-language-code-update.mdx)

Update language code in the existing checkout.
