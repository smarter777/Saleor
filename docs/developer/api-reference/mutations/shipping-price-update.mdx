Updates a new shipping price.

```graphql
shippingPriceUpdate(
  id: ID!
  input: ShippingPriceInput!
): ShippingPriceUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a shipping price to update.

#### `input` ([`ShippingPriceInput!`](/developer/api-reference/inputs/shipping-price-input.mdx))

Fields required to update a shipping price.

### Type

#### [`ShippingPriceUpdate`](/developer/api-reference/objects/shipping-price-update.mdx)

Updates a new shipping price.
