Updates a collection.

```graphql
collectionUpdate(
  id: ID!
  input: CollectionInput!
): CollectionUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a collection to update.

#### `input` ([`CollectionInput!`](/developer/api-reference/inputs/collection-input.mdx))

Fields required to update a collection.

### Type

#### [`CollectionUpdate`](/developer/api-reference/objects/collection-update.mdx)

Updates a collection.
