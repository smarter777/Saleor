Creates a new menu item.

Requires one of the following permissions: MANAGE_MENUS.

```graphql
menuItemCreate(
  input: MenuItemCreateInput!
): MenuItemCreate
```

### Arguments

#### [`input`](#) ([`MenuItemCreateInput!`](../../../developer/api-reference/inputs/menu-item-create-input))

Fields required to update a menu item. Only one of `url`, `category`, `page`, `collection` is allowed per item.

### Type

#### [`MenuItemCreate`](../../../developer/api-reference/objects/menu-item-create)

Creates a new menu item.

Requires one of the following permissions: MANAGE_MENUS.
