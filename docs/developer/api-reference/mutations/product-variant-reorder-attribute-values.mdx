Reorder product variant attribute values.

```graphql
productVariantReorderAttributeValues(
  attributeId: ID!
  moves: [ReorderInput]!
  variantId: ID!
): ProductVariantReorderAttributeValues

```

### Arguments

#### `attributeId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an attribute.

#### `moves` ([`[ReorderInput]!`](/developer/api-reference/inputs/reorder-input.mdx))

The list of reordering operations for given attribute values.

#### `variantId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product variant.

### Type

#### [`ProductVariantReorderAttributeValues`](/developer/api-reference/objects/product-variant-reorder-attribute-values.mdx)

Reorder product variant attribute values.
