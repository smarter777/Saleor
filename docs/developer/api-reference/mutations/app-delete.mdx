Deletes an app.

```graphql
appDelete(
  id: ID!
): AppDelete

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an app to delete.

### Type

#### [`AppDelete`](/developer/api-reference/objects/app-delete.mdx)

Deletes an app.
