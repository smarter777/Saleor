Deletes an authentication token assigned to app.

```graphql
appTokenDelete(
  id: ID!
): AppTokenDelete

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an auth token to delete.

### Type

#### [`AppTokenDelete`](/developer/api-reference/objects/app-token-delete.mdx)

Deletes an authentication token assigned to app.
