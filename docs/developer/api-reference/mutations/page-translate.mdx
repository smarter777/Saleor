Creates/Updates translations for Page.

```graphql
pageTranslate(
  id: ID!
  input: PageTranslationInput!
  languageCode: LanguageCodeEnum!
): PageTranslate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

Page ID.

#### `input` ([`PageTranslationInput!`](/developer/api-reference/inputs/page-translation-input.mdx))

#### `languageCode` ([`LanguageCodeEnum!`](/developer/api-reference/enums/language-code-enum.mdx))

Translation language code.

### Type

#### [`PageTranslate`](/developer/api-reference/objects/page-translate.mdx)

Creates/Updates translations for Page.
