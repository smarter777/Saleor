Creates product variants for a given product.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
productVariantBulkCreate(
  product: ID!
  variants: [ProductVariantBulkCreateInput!]!
): ProductVariantBulkCreate
```
