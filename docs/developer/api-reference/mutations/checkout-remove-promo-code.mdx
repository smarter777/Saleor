Remove a gift card or a voucher from a checkout.

```graphql
checkoutRemovePromoCode(
  checkoutId: ID!
  promoCode: String!
): CheckoutRemovePromoCode

```

### Arguments

#### `checkoutId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

Checkout ID.

#### `promoCode` ([`String!`](/developer/api-reference/scalars/string.mdx))

Gift card code or voucher code.

### Type

#### [`CheckoutRemovePromoCode`](/developer/api-reference/objects/checkout-remove-promo-code.mdx)

Remove a gift card or a voucher from a checkout.
