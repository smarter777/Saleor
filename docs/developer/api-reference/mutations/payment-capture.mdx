Captures the authorized payment amount.

Requires one of the following permissions: MANAGE_ORDERS.

```graphql
paymentCapture(
  amount: PositiveDecimal
  paymentId: ID!
): PaymentCapture
```

#### [`paymentId`](#) ([`ID!`](../../../developer/api-reference/scalars/id))
