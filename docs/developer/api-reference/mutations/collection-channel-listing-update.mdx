Manage collection's availability in channels.

Requires one of the following permissions: MANAGE_PRODUCTS.

```graphql
collectionChannelListingUpdate(
  id: ID!
  input: CollectionChannelListingUpdateInput!
): CollectionChannelListingUpdate
```

### Arguments

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

ID of a collection to update.

#### [`input`](#) ([`CollectionChannelListingUpdateInput!`](../../../developer/api-reference/inputs/collection-channel-listing-update-input))

Fields required to create or update collection channel listings.
