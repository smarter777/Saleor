---

Add shipping zone to given warehouse.

```graphql
assignWarehouseShippingZone(
  id: ID!
  shippingZoneIds: [ID!]!
): WarehouseShippingZoneAssign

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a warehouse to update.

#### `shippingZoneIds` ([`[ID!]!`](/developer/api-reference/scalars/id.mdx))

List of shipping zone IDs.

### Type

#### [`WarehouseShippingZoneAssign`](/developer/api-reference/objects/warehouse-shipping-zone-assign.mdx)

Add shipping zone to given warehouse.
