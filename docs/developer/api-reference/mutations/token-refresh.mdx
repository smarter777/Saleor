Refresh JWT token. Mutation tries to take refreshToken from the input.If it fails it will try to take refreshToken from the http-only cookie -refreshToken. csrfToken is required when refreshToken is provided as a cookie.

```graphql
tokenRefresh(
  csrfToken: String
  refreshToken: String
): RefreshToken

```

### Arguments

#### `csrfToken` ([`String`](/developer/api-reference/scalars/string.mdx))

CSRF token required to refresh token. This argument is required when refreshToken is provided as a cookie.

#### `refreshToken` ([`String`](/developer/api-reference/scalars/string.mdx))

Refresh token.

### Type

#### [`RefreshToken`](/developer/api-reference/objects/refresh-token.mdx)

Refresh JWT token. Mutation tries to take refreshToken from the input.If it fails it will try to take refreshToken from the http-only cookie -refreshToken. csrfToken is required when refreshToken is provided as a cookie.
