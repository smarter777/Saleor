Prepare external authentication url for user by custom plugin.

```graphql
externalAuthenticationUrl(
  input: JSONString!
  pluginId: String!
): ExternalAuthenticationUrl

```

### Arguments

#### `input` ([`JSONString!`](/developer/api-reference/scalars/json-string.mdx))

The data required by plugin to create external authentication url.

#### `pluginId` ([`String!`](/developer/api-reference/scalars/string.mdx))

The ID of the authentication plugin.

### Type

#### [`ExternalAuthenticationUrl`](/developer/api-reference/objects/external-authentication-url.mdx)

Prepare external authentication url for user by custom plugin.
