Deletes a value of an attribute.

```graphql
attributeValueDelete(
  id: ID!
): AttributeValueDelete

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a value to delete.

### Type

#### [`AttributeValueDelete`](/developer/api-reference/objects/attribute-value-delete.mdx)

Deletes a value of an attribute.
