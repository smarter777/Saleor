Verify external authentication data by plugin.

```graphql
externalVerify(
  input: JSONString!
  pluginId: String!
): ExternalVerify

```

### Arguments

#### `input` ([`JSONString!`](/developer/api-reference/scalars/json-string.mdx))

The data required by plugin to proceed the verification.

#### `pluginId` ([`String!`](/developer/api-reference/scalars/string.mdx))

The ID of the authentication plugin.

### Type

#### [`ExternalVerify`](/developer/api-reference/objects/external-verify.mdx)

Verify external authentication data by plugin.
