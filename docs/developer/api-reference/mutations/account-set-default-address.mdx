Sets a default address for the authenticated user.

```graphql
accountSetDefaultAddress(
  id: ID!
  type: AddressTypeEnum!
): AccountSetDefaultAddress

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the address to set as default.

#### `type` ([`AddressTypeEnum!`](/developer/api-reference/enums/address-type-enum.mdx))

The type of address.

### Type

#### [`AccountSetDefaultAddress`](/developer/api-reference/objects/account-set-default-address.mdx)

Sets a default address for the authenticated user.
