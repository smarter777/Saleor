Updates a product image.

```graphql
productImageUpdate(
  id: ID!
  input: ProductImageUpdateInput!
): ProductImageUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product image to update.

#### `input` ([`ProductImageUpdateInput!`](/developer/api-reference/inputs/product-image-update-input.mdx))

Fields required to update a product image.

### Type

#### [`ProductImageUpdate`](/developer/api-reference/objects/product-image-update.mdx)

Updates a product image.
