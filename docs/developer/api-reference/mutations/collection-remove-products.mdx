Remove products from a collection.

```graphql
collectionRemoveProducts(
  collectionId: ID!
  products: [ID]!
): CollectionRemoveProducts

```

### Arguments

#### `collectionId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a collection.

#### `products` ([`[ID]!`](/developer/api-reference/scalars/id.mdx))

List of product IDs.

### Type

#### [`CollectionRemoveProducts`](/developer/api-reference/objects/collection-remove-products.mdx)

Remove products from a collection.
