Create order lines for a draft order.

```graphql
draftOrderLinesCreate(
  id: ID!
  input: [OrderLineCreateInput]!
): DraftOrderLinesCreate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of the draft order to add the lines to.

#### `input` ([`[OrderLineCreateInput]!`](/developer/api-reference/inputs/order-line-create-input.mdx))

Fields required to add order lines.

### Type

#### [`DraftOrderLinesCreate`](/developer/api-reference/objects/draft-order-lines-create.mdx)

Create order lines for a draft order.
