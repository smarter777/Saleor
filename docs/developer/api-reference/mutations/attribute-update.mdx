Updates attribute.

```graphql
attributeUpdate(
  id: ID!
  input: AttributeUpdateInput!
): AttributeUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of an attribute to update.

#### `input` ([`AttributeUpdateInput!`](/developer/api-reference/inputs/attribute-update-input.mdx))

Fields required to update an attribute.

### Type

#### [`AttributeUpdate`](/developer/api-reference/objects/attribute-update.mdx)

Updates attribute.
