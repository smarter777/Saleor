Updates a webhook subscription.

```graphql
webhookUpdate(
  id: ID!
  input: WebhookUpdateInput!
): WebhookUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a webhook to update.

#### `input` ([`WebhookUpdateInput!`](/developer/api-reference/inputs/webhook-update-input.mdx))

Fields required to update a webhook.

### Type

#### [`WebhookUpdate`](/developer/api-reference/objects/webhook-update.mdx)

Updates a webhook subscription.
