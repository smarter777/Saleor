Update plugin configuration.

```graphql
pluginUpdate(
  id: ID!
  input: PluginUpdateInput!
): PluginUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of plugin to update.

#### `input` ([`PluginUpdateInput!`](/developer/api-reference/inputs/plugin-update-input.mdx))

Fields required to update a plugin configuration.

### Type

#### [`PluginUpdate`](/developer/api-reference/objects/plugin-update.mdx)

Update plugin configuration.
