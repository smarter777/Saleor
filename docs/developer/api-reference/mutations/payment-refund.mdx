Refunds the captured payment amount.

```graphql
paymentRefund(
  amount: PositiveDecimal
  paymentId: ID!
): PaymentRefund

```

### Arguments

#### `amount` ([`PositiveDecimal`](/developer/api-reference/scalars/positive-decimal.mdx))

Transaction amount.

#### `paymentId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

Payment ID.

### Type

#### [`PaymentRefund`](/developer/api-reference/objects/payment-refund.mdx)

Refunds the captured payment amount.
