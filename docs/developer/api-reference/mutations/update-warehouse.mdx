Updates given warehouse.

```graphql
updateWarehouse(
  id: ID!
  input: WarehouseUpdateInput!
): WarehouseUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a warehouse to update.

#### `input` ([`WarehouseUpdateInput!`](/developer/api-reference/inputs/warehouse-update-input.mdx))

Fields required to update warehouse.

### Type

#### [`WarehouseUpdate`](/developer/api-reference/objects/warehouse-update.mdx)

Updates given warehouse.
