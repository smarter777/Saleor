Request email change of the logged in user.

```graphql
requestEmailChange(
  newEmail: String!
  password: String!
  redirectUrl: String!
): RequestEmailChange

```

### Arguments

#### `newEmail` ([`String!`](/developer/api-reference/scalars/string.mdx))

New user email.

#### `password` ([`String!`](/developer/api-reference/scalars/string.mdx))

User password.

#### `redirectUrl` ([`String!`](/developer/api-reference/scalars/string.mdx))

URL of a view where users should be redirected to update the email address. URL in RFC 1808 format.

### Type

#### [`RequestEmailChange`](/developer/api-reference/objects/request-email-change.mdx)

Request email change of the logged in user.
