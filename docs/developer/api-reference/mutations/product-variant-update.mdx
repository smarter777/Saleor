Updates an existing variant for product.

```graphql
productVariantUpdate(
  id: ID!
  input: ProductVariantInput!
): ProductVariantUpdate

```

### Arguments

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product variant to update.

#### `input` ([`ProductVariantInput!`](/developer/api-reference/inputs/product-variant-input.mdx))

Fields required to update a product variant.

### Type

#### [`ProductVariantUpdate`](/developer/api-reference/objects/product-variant-update.mdx)

Updates an existing variant for product.
