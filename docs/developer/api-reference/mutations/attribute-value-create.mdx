Creates a value for an attribute.

```graphql
attributeValueCreate(
  attribute: ID!
  input: AttributeValueCreateInput!
): AttributeValueCreate

```

### Arguments

#### `attribute` ([`ID!`](/developer/api-reference/scalars/id.mdx))

Attribute to which value will be assigned.

#### `input` ([`AttributeValueCreateInput!`](/developer/api-reference/inputs/attribute-value-create-input.mdx))

Fields required to create an AttributeValue.

### Type

#### [`AttributeValueCreate`](/developer/api-reference/objects/attribute-value-create.mdx)

Creates a value for an attribute.
