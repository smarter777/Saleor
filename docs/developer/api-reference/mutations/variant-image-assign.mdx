Assign an image to a product variant.

```graphql
variantImageAssign(
  imageId: ID!
  variantId: ID!
): VariantImageAssign

```

### Arguments

#### `imageId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product image to assign to a variant.

#### `variantId` ([`ID!`](/developer/api-reference/scalars/id.mdx))

ID of a product variant.

### Type

#### [`VariantImageAssign`](/developer/api-reference/objects/variant-image-assign.mdx)

Assign an image to a product variant.
