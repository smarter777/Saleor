---


New in Saleor 3.2. Represents a payment transaction. Note: this feature is in a preview state and can be subject to changes at later point.

```graphql
type TransactionItem implements Node, ObjectWithMetadata {
  id: ID!
  privateMetadata: [MetadataItem!]!
  metadata: [MetadataItem!]!
  createdAt: DateTime!
  modifiedAt: DateTime!
  actions: [TransactionActionEnum!]!
  authorizedAmount: Money!
  refundedAmount: Money!
  voidedAmount: Money!
  capturedAmount: Money!
  status: String!
  type: String!
  reference: String!
  events: [TransactionEvent!]!
}
```


### Fields

#### `id` ([`ID`](../../../developer/api-reference/scalars/id))

The ID of the object.


#### `privateMetadata` ([`MetadataItem`](../../../developer/api-reference/objects/metadata-item))

List of private metadata items. Requires proper staff permissions to access.


#### `metadata` ([`MetadataItem`](../../../developer/api-reference/objects/metadata-item))

List of public metadata items. Can be accessed without permissions.


#### `createdAt` ([`DateTime`](../../../developer/api-reference/scalars/date-time))




#### `modifiedAt` ([`DateTime`](../../../developer/api-reference/scalars/date-time))




#### `actions` ([`TransactionActionEnum`](../../../developer/api-reference/enums/transaction-action-enum))

List of actions that can be performed in the current state of a payment.


#### `authorizedAmount` ([`Money`](../../../developer/api-reference/objects/money))

Total amount authorized for this payment.


#### `refundedAmount` ([`Money`](../../../developer/api-reference/objects/money))

Total amount refunded for this payment.


#### `voidedAmount` ([`Money`](../../../developer/api-reference/objects/money))

Total amount voided for this payment.


#### `capturedAmount` ([`Money`](../../../developer/api-reference/objects/money))

Total amount captured for this payment.


#### `status` ([`String`](../../../developer/api-reference/scalars/string))

Status of transaction.


#### `type` ([`String`](../../../developer/api-reference/scalars/string))

Type of transaction.


#### `reference` ([`String`](../../../developer/api-reference/scalars/string))

Reference of transaction.


#### `events` ([`TransactionEvent`](../../../developer/api-reference/objects/transaction-event))

List of all transaction's events.
