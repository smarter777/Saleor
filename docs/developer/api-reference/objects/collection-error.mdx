No description

```graphql
type CollectionError {
  field: String
  message: String
  products: [ID!]
  code: CollectionErrorCode!
}
```

### Fields

#### `field` ([`String`](/developer/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/developer/api-reference/scalars/string.mdx))

The error message.

#### `products` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of products IDs which causes the error.

#### `code` ([`CollectionErrorCode!`](/developer/api-reference/enums/collection-error-code.mdx))

The error code.
