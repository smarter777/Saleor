Represents a reduced VAT rate for a particular type of goods.

```graphql
type ReducedRate {
  rate: Float!
  rateType: TaxRateType!
}
```

### Fields

#### `rate` ([`Float!`](/developer/api-reference/scalars/float.mdx))

Reduced VAT rate in percent.

#### `rateType` ([`TaxRateType!`](/developer/api-reference/enums/tax-rate-type.mdx))

A type of goods.
