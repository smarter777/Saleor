```graphql
type BulkProductError {
  field: String
  message: String
  code: ProductErrorCode!
  attributes: [ID!]
  values: [ID!]
  index: Int
  warehouses: [ID!]
  channels: [ID!]
}
```

### Fields

#### `field` ([`String`](/developer/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/developer/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`ProductErrorCode!`](/developer/api-reference/enums/product-error-code.mdx))

The error code.

#### `attributes` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of attributes IDs which causes the error.

#### `values` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of attribute values IDs which causes the error.

#### `index` ([`Int`](/developer/api-reference/scalars/int.mdx))

Index of an input list item that caused the error.

#### `warehouses` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of warehouse IDs which causes the error.

#### `channels` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of channel IDs which causes the error.
