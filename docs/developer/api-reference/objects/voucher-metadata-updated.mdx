Added in Saleor 3.8.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

```graphql
type VoucherMetadataUpdated implements Event {
  issuedAt: DateTime
  version: String
  issuingPrincipal: IssuingPrincipal
  recipient: App
  voucher(
  channel: String
): Voucher
}
```
