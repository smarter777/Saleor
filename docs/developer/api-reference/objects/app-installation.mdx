Represents ongoing installation of app.

```graphql
type AppInstallation implements Node, Job {
  appName: String!
  manifestUrl: String!
  id: ID!
  status: JobStatusEnum!
  createdAt: DateTime!
  updatedAt: DateTime!
  message: String
}
```
