Represents shop's domain.

```graphql
type Domain {
  host: String!
  sslEnabled: Boolean!
  url: String!
}
```

### Fields

#### `host` ([`String!`](/developer/api-reference/scalars/string.mdx))

The host name of the domain.

#### `sslEnabled` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

Inform if SSL is enabled.

#### `url` ([`String!`](/developer/api-reference/scalars/string.mdx))

Shop's absolute URL.
