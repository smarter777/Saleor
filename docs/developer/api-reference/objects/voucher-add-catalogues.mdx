Adds products, categories, collections to a voucher.

Requires one of the following permissions: MANAGE_DISCOUNTS.

```graphql
type VoucherAddCatalogues {
  voucher: Voucher
  discountErrors: [DiscountError!]!
  errors: [DiscountError!]!
}
```
