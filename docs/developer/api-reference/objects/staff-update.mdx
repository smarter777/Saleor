Updates an existing staff user. Apps are not allowed to perform this mutation.

Requires one of the following permissions: MANAGE_STAFF.

```graphql
type StaffUpdate {
  staffErrors: [StaffError!]!
  errors: [StaffError!]!
  user: User
}
```
