Represents order fulfillment.

```graphql
type Fulfillment implements Node, ObjectWithMetadata {
  id: ID!
  fulfillmentOrder: Int!
  status: FulfillmentStatus!
  trackingNumber: String!
  created: DateTime!
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  lines: [FulfillmentLine]
  statusDisplay: String
  warehouse: Warehouse
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `fulfillmentOrder` ([`Int!`](/developer/api-reference/scalars/int.mdx))

#### `status` ([`FulfillmentStatus!`](/developer/api-reference/enums/fulfillment-status.mdx))

#### `trackingNumber` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `created` ([`DateTime!`](/developer/api-reference/scalars/date-time.mdx))

#### `privateMetadata` ([`[MetadataItem]!`](/developer/api-reference/objects/metadata-item.mdx))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](/developer/api-reference/objects/metadata-item.mdx))

List of public metadata items. Can be accessed without permissions.

#### `lines` ([`[FulfillmentLine]`](/developer/api-reference/objects/fulfillment-line.mdx))

List of lines for the fulfillment.

#### `statusDisplay` ([`String`](/developer/api-reference/scalars/string.mdx))

User-friendly fulfillment status.

#### `warehouse` ([`Warehouse`](/developer/api-reference/objects/warehouse.mdx))

Warehouse from fulfillment was fulfilled.
