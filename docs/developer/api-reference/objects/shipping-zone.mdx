title: ShippingZone
---

Represents a shipping zone in the shop. Zones are the concept used only for grouping shipping methods in the dashboard, and are never exposed to the customers directly.

```graphql
type ShippingZone implements Node, ObjectWithMetadata {
  id: ID!
  name: String!
  default: Boolean!
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  priceRange: MoneyRange
  countries: [CountryDisplay]
  shippingMethods: [ShippingMethod]
  warehouses: [Warehouse!]!
  channels: [Channel!]!
  description: String
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `name` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `default` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

#### `privateMetadata` ([`[MetadataItem]!`](/developer/api-reference/objects/metadata-item.mdx))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](/developer/api-reference/objects/metadata-item.mdx))

List of public metadata items. Can be accessed without permissions.

#### `priceRange` ([`MoneyRange`](/developer/api-reference/objects/money-range.mdx))

Lowest and highest prices for the shipping.

#### `countries` ([`[CountryDisplay]`](/developer/api-reference/objects/country-display.mdx))

List of countries available for the method.

#### `shippingMethods` ([`[ShippingMethod]`](/developer/api-reference/objects/shipping-method.mdx))

List of shipping methods available for orders shipped to countries within this shipping zone.

#### `warehouses` ([`[Warehouse!]!`](/developer/api-reference/objects/warehouse.mdx))

List of warehouses for shipping zone.

#### `channels` ([`[Channel!]!`](/developer/api-reference/objects/channel.mdx))

List of channels for shipping zone.

#### `description` ([`String`](/developer/api-reference/scalars/string.mdx))

Description of a shipping zone.
