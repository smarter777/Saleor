```graphql
type ChannelError {
  field: String
  message: String
  code: ChannelErrorCode!
  shippingZones: [ID!]
}
```

### Fields

#### `field` ([`String`](/developer/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/developer/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`ChannelErrorCode!`](/developer/api-reference/enums/channel-error-code.mdx))

The error code.

#### `shippingZones` ([`[ID!]`](/developer/api-reference/scalars/id.mdx))

List of shipping zone IDs which causes the error.
