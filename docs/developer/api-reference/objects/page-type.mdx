title: PageType
---

Represents a type of page. It defines what attributes are available to pages of this type.

```graphql
type PageType implements Node, ObjectWithMetadata {
  id: ID!
  name: String!
  slug: String!
  privateMetadata: [MetadataItem]!
  metadata: [MetadataItem]!
  attributes: [Attribute]
  availableAttributes(
    filter: AttributeFilterInput
    before: String
    after: String
    first: Int = 0
    last: Int = 0
  ): AttributeCountableConnection
  hasPages: Boolean
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `name` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `slug` ([`String!`](/developer/api-reference/scalars/string.mdx))

#### `privateMetadata` ([`[MetadataItem]!`](/developer/api-reference/objects/metadata-item.mdx))

List of private metadata items.Requires proper staff permissions to access.

#### `metadata` ([`[MetadataItem]!`](/developer/api-reference/objects/metadata-item.mdx))

List of public metadata items. Can be accessed without permissions.

#### `attributes` ([`[Attribute]`](/developer/api-reference/objects/attribute.mdx))

Page attributes of that page type.

#### `availableAttributes` ([`AttributeCountableConnection`](/developer/api-reference/objects/attribute-countable-connection.mdx))

Attributes that can be assigned to the page type.

- ##### `filter` ([`AttributeFilterInput`](/developer/api-reference/inputs/attribute-filter-input.mdx))

- ##### `before` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come before the specified cursor.

- ##### `after` ([`String`](/developer/api-reference/scalars/string.mdx))

Return the elements in the list that come after the specified cursor.

- ##### `first` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the first n elements from the list.

- ##### `last` ([`Int`](/developer/api-reference/scalars/int.mdx))

Return the last n elements from the list.

#### `hasPages` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Whether page type has pages assigned.
