```graphql
type TranslationError {
  field: String
  message: String
  code: TranslationErrorCode!
}
```

### Fields

#### `field` ([`String`](/developer/api-reference/scalars/string.mdx))

Name of a field that caused the error. A value of `null` indicates that the error isn't associated with a particular field.

#### `message` ([`String`](/developer/api-reference/scalars/string.mdx))

The error message.

#### `code` ([`TranslationErrorCode!`](/developer/api-reference/enums/translation-error-code.mdx))

The error code.
