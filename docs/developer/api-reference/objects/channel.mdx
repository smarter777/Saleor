Represents channel.

```graphql
type Channel implements Node {
  id: ID!
  slug: String!
  name: String!
  isActive: Boolean!
  currencyCode: String!
  hasOrders: Boolean!
  defaultCountry: CountryDisplay!
  warehouses: [Warehouse!]!
  countries: [CountryDisplay!]
  availableShippingMethodsPerCountry(
    countries: [CountryCode!]
  ): [ShippingMethodsPerCountry!]
  stockSettings: StockSettings!
}
```

#### [`currencyCode`](#) ([`String!`](../../../developer/api-reference/scalars/string))

#### [`hasOrders`](#) ([`Boolean!`](../../../developer/api-reference/scalars/boolean))

Whether a channel has associated orders.

Requires one of the following permissions: MANAGE_CHANNELS.

#### [`defaultCountry`](#) ([`CountryDisplay!`](../../../developer/api-reference/objects/country-display))

Default country for the channel. Default country can be used in checkout to determine the stock quantities or calculate taxes when the country was not explicitly provided.

Added in Saleor 3.1.

Requires one of the following permissions: AUTHENTICATED_APP, AUTHENTICATED_STAFF_USER.

#### [`warehouses`](#) ([`[Warehouse!]!`](../../../developer/api-reference/objects/warehouse))

List of warehouses assigned to this channel.

Added in Saleor 3.5.

Note: this API is currently in Feature Preview and can be subject to changes at later point.

Requires one of the following permissions: AUTHENTICATED_APP, AUTHENTICATED_STAFF_USER.

#### [`countries`](#) ([`[CountryDisplay!]`](../../../developer/api-reference/objects/country-display))

List of shippable countries for the channel.

Added in Saleor 3.6.

Note: this API is currently in Feature Preview and can be subject to changes at later point.
