Deletes staff users. Apps are not allowed to perform this mutation.

Requires one of the following permissions: MANAGE_STAFF.

```graphql
type StaffBulkDelete {
  count: Int!
  staffErrors: [StaffError!]!
  errors: [StaffError!]!
}
```
