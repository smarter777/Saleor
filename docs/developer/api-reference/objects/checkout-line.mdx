Represents an item in the checkout.

```graphql
type CheckoutLine implements Node {
  id: ID!
  variant: ProductVariant!
  quantity: Int!
  totalPrice: TaxedMoney
  requiresShipping: Boolean
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `variant` ([`ProductVariant!`](/developer/api-reference/objects/product-variant.mdx))

#### `quantity` ([`Int!`](/developer/api-reference/scalars/int.mdx))

#### `totalPrice` ([`TaxedMoney`](/developer/api-reference/objects/taxed-money.mdx))

The sum of the checkout line price, taxes and discounts.

#### `requiresShipping` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Indicates whether the item need to be delivered.

### Interfaces

#### [`Node`](/developer/api-reference/interfaces/node.mdx)

An object with an ID
