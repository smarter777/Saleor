Represents a product image.

```graphql
type ProductImage {
  id: ID!
  alt: String
  sortOrder: Int
  url(size: Int, format: ThumbnailFormatEnum): String!
}
```

### Fields

#### [`id`](#) ([`ID!`](../../../developer/api-reference/scalars/id))

The ID of the image.

#### [`alt`](#) ([`String`](../../../developer/api-reference/scalars/string))

The alt text of the image.

#### [`sortOrder`](#) ([`Int`](../../../developer/api-reference/scalars/int))

The new relative sorting position of the item (from -inf to +inf). 1 moves the item one position forward, -1 moves the item one position backward, 0 leaves the item unchanged.

#### [`url`](#) ([`String!`](../../../developer/api-reference/scalars/string))

- ##### [`size`](#) ([`Int`](../../../developer/api-reference/scalars/int))

Size of the image. If not provided, the original image will be returned.

- ##### [`format`](#) ([`ThumbnailFormatEnum`](../../../developer/api-reference/enums/thumbnail-format-enum))

The format of the image. When not provided, format of the original image will be used. Must be provided together with the size value, otherwise original image will be returned.

Added in Saleor 3.6.

Note: this API is currently in Feature Preview and can be subject to changes at later point.
