---

Stores information about a single configuration field.

```graphql
type ConfigurationItem {
  name: String!
  value: String
  type: ConfigurationTypeFieldEnum
  helpText: String
  label: String
}
```

### Fields

#### `name` ([`String!`](/developer/api-reference/scalars/string.mdx))

Name of the field.

#### `value` ([`String`](/developer/api-reference/scalars/string.mdx))

Current value of the field.

#### `type` ([`ConfigurationTypeFieldEnum`](/developer/api-reference/enums/configuration-type-field-enum.mdx))

Type of the field.

#### `helpText` ([`String`](/developer/api-reference/scalars/string.mdx))

Help text for the field.

#### `label` ([`String`](/developer/api-reference/scalars/string.mdx))

Label for the field.
