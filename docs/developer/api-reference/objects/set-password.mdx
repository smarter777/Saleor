Sets the user's password from the token sent by email using the RequestPasswordReset mutation.

```graphql
type SetPassword {
  errors: [Error!]!
  token: String
  refreshToken: String
  csrfToken: String
  user: User
  accountErrors: [AccountError!]!
}
```

### Fields

#### `errors` ([`[Error!]!`](/developer/api-reference/objects/error.mdx))

<sub><sup><span className="deprecatedTag">DEPRECATED</span> Use typed errors with error codes. This field will be removed after 2020-07-31.</sup></sub>

List of errors that occurred executing the mutation.

#### `token` ([`String`](/developer/api-reference/scalars/string.mdx))

JWT token, required to authenticate.

#### `refreshToken` ([`String`](/developer/api-reference/scalars/string.mdx))

JWT refresh token, required to re-generate access token.

#### `csrfToken` ([`String`](/developer/api-reference/scalars/string.mdx))

CSRF token required to re-generate access token.

#### `user` ([`User`](/developer/api-reference/objects/user.mdx))

A user instance.

#### `accountErrors` ([`[AccountError!]!`](/developer/api-reference/objects/account-error.mdx))
