A gift card is a prepaid electronic payment card accepted in stores. They can be used during checkout by providing a valid gift card codes.

```graphql
type GiftCard implements Node {
  code: String
  user: User
  created: DateTime!
  startDate: Date!
  endDate: Date
  lastUsedOn: DateTime
  isActive: Boolean!
  initialBalance: Money
  currentBalance: Money
  id: ID!
  displayCode: String
}
```

### Fields

#### `code` ([`String`](/developer/api-reference/scalars/string.mdx))

Gift card code.

#### `user` ([`User`](/developer/api-reference/objects/user.mdx))

#### `created` ([`DateTime!`](/developer/api-reference/scalars/date-time.mdx))

#### `endDate` ([`Date`](/developer/api-reference/scalars/date.mdx))

#### `isActive` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

#### `initialBalance` ([`Money`](/developer/api-reference/objects/money.mdx))

#### `currentBalance` ([`Money`](/developer/api-reference/objects/money.mdx))

The ID of the object.

#### `displayCode` ([`String`](/developer/api-reference/scalars/string.mdx))
