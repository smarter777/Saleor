Represents product channel listing.

```graphql
type ProductChannelListing implements Node {
  id: ID!
  publicationDate: Date
  isPublished: Boolean!
  channel: Channel!
  visibleInListings: Boolean!
  availableForPurchase: Date
  discountedPrice: Money
  purchaseCost: MoneyRange
  margin: Margin
  isAvailableForPurchase: Boolean
  pricing: ProductPricingInfo
}
```

### Fields

#### `id` ([`ID!`](/developer/api-reference/scalars/id.mdx))

The ID of the object.

#### `publicationDate` ([`Date`](/developer/api-reference/scalars/date.mdx))

#### `isPublished` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

#### `channel` ([`Channel!`](/developer/api-reference/objects/channel.mdx))

#### `visibleInListings` ([`Boolean!`](/developer/api-reference/scalars/boolean.mdx))

#### `availableForPurchase` ([`Date`](/developer/api-reference/scalars/date.mdx))

#### `discountedPrice` ([`Money`](/developer/api-reference/objects/money.mdx))

The price of the cheapest variant (including discounts).

#### `purchaseCost` ([`MoneyRange`](/developer/api-reference/objects/money-range.mdx))

Purchase cost of product.

#### `margin` ([`Margin`](/developer/api-reference/objects/margin.mdx))

Range of margin percentage value.

#### `isAvailableForPurchase` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Whether the product is available for purchase.

#### `pricing` ([`ProductPricingInfo`](/developer/api-reference/objects/product-pricing-info.mdx))

Lists the storefront product's pricing, the current price and discounts, only meant for displaying.
