Represents a custom attribute.

```graphql
type SelectedAttribute {
  attribute: Attribute!
  values: [AttributeValue]!
}
```

### Fields

#### `attribute` ([`Attribute!`](/developer/api-reference/objects/attribute.mdx))

Name of an attribute displayed in the interface.

#### `values` ([`[AttributeValue]!`](/developer/api-reference/objects/attribute-value.mdx))

Values of an attribute.
