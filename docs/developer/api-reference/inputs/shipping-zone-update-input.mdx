```graphql
type ShippingZoneUpdateInput {
```

### Fields

#### `name` ([`String`](/developer/api-reference/scalars/string.mdx))

Shipping zone's name. Visible only to the staff.

#### `description` ([`String`](/developer/api-reference/scalars/string.mdx))

Description of the shipping zone.

#### `countries` ([`[String]`](/developer/api-reference/scalars/string.mdx))

List of countries in this shipping zone.

#### `default` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Default shipping zone will be used for countries not covered by other zones.

#### `addWarehouses` ([`[ID]`](/developer/api-reference/scalars/id.mdx))

List of warehouses to assign to a shipping zone

#### `removeWarehouses` ([`[ID]`](/developer/api-reference/scalars/id.mdx))

List of warehouses to unassign from a shipping zone
