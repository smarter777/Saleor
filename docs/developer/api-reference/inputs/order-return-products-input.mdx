```graphql
type OrderReturnProductsInput {

  fulfillmentLines: [OrderReturnFulfillmentLineInput!]

}
```

### Fields

#### `orderLines` ([`[OrderReturnLineInput!]`](/developer/api-reference/inputs/order-return-line-input.mdx))

List of unfulfilled lines to return.

#### `fulfillmentLines` ([`[OrderReturnFulfillmentLineInput!]`](/developer/api-reference/inputs/order-return-fulfillment-line-input.mdx))

List of fulfilled lines to return.

#### `amountToRefund` ([`PositiveDecimal`](/developer/api-reference/scalars/positive-decimal.mdx))

The total amount of refund when the value is provided manually.

#### `includeShippingCosts` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

If true, Saleor will refund shipping costs. If amountToRefund is providedincludeShippingCosts will be ignored.

#### `refund` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

If true, Saleor will call refund action for all lines.
