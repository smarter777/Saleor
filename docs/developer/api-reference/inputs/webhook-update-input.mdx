No description

```graphql
type WebhookUpdateInput {
  name: String
  targetUrl: String
  events: [WebhookEventTypeEnum]
  app: ID
  isActive: Boolean
  secretKey: String
}
```

### Fields

#### `name` ([`String`](/developer/api-reference/scalars/string.mdx))

The new name of the webhook.

#### `targetUrl` ([`String`](/developer/api-reference/scalars/string.mdx))

The url to receive the payload.

#### `events` ([`[WebhookEventTypeEnum]`](/developer/api-reference/enums/webhook-event-type-enum.mdx))

The events that webhook wants to subscribe. The CHECKOUT_QUANTITY_CHANGED is deprecated. It will be removed in Saleor 3.0

#### `app` ([`ID`](/developer/api-reference/scalars/id.mdx))

ID of the app to which webhook belongs.

#### `isActive` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Determine if webhook will be set active or not.

#### `secretKey` ([`String`](/developer/api-reference/scalars/string.mdx))

Use to create a hash signature with each payload.
