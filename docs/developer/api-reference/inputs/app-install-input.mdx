### Fields

#### `appName` ([`String`](/developer/api-reference/scalars/string.mdx))

Url to app's manifest in JSON format.

#### `activateAfterInstallation` ([`Boolean`](/developer/api-reference/scalars/boolean.mdx))

Determine if app will be set active or not.

#### `permissions` ([`[PermissionEnum]`](/developer/api-reference/enums/permission-enum.mdx))

List of permission code names to assign to this app.
