## Overview

Saleor is an open-source project which means all its code is [publically available](https://github.com/saleor). That gives you two ways you can host it:

- Self-hosting - you host and maintain Saleor yourself.
- [Saleor Cloud](https://saleor.io/pricing/) - we do it for you.

## Self-hosting

If you are interested in hosting Saleor yourself, please continue reading the documentation in this chapter. In the next articles, you will learn about:

## [Saleor Cloud](https://saleor.io/pricing/)

Saleor Cloud is a stable and scalable serverless environment for hosting Saleor. It allows you to use all its best features without the technical overhead.

Explore its pricing and advantages on the [Saleor Cloud page](https://saleor.io/pricing/).

:::tip
The fastest way to try out Saleor is by [utilizing a free developer plan in Saleor Cloud](https://cloud.saleor.io/signup).
:::
