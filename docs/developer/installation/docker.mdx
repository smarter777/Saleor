Using Docker to build software allows you to run and test code without having to worry about external dependencies such as cache servers and databases.

:::info
The following setup is only meant for local development. See [Docker Deployment](../deployment/docker.md) for production use.
:::

## Prerequisites

You will need to install [Docker Desktop](https://www.docker.com/products/docker-desktop) and [Docker Compose](https://docs.docker.com/compose/install/) before following the instructions below.

:::caution
At the time of this writing, Docker Desktop is not available for Windows Home. Please use the [manual installation instructions](windows.mdx) if you can't install Docker.
:::

## Installation

Please follow the instructions provided by the [saleor-platform](https://github.com/mirumee/saleor-platform/) repository.

## Troubleshooting

Make sure Docker and Docker Compose work before attempting to run Saleor. We won't be able to assist with your Docker installation.

If you encounter other problems, you can ask for help in our [Spectrum community](https://spectrum.chat/saleor).
