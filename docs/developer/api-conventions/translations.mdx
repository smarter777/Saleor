## Introduction

Translating shop consist of two steps:
- translating storefront application
- translating products and other data available in API

Frontend part can be managed using extensions like [Format.JS](https://formatjs.io). Generated files can be later edited by hand or using external service, for example [Transifex](https://www.transifex.com) or [POEditor](https://poeditor.com). Besides text, prices and dates also should be localized using [`Intl.NumberFormat`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/NumberFormat) and [date-fns](https://date-fns.org).

For translating database data Saleor delivers specialized translation objects which can be modified in dashboard or via API.

## Permissions
All users having `MANAGE_TRANSLATIONS` can change and create translations. Additional permissions for managing objects are not needed.
